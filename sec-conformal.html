    Section 3.5 Conformal mappings

    The idea is that a conformal map should be one that “preserves angles”. To state this precisely, we will need tangent vectors.

    Definition 3.5.1.

    Let \(\gamma_1,\gamma_2\colon[-1,1]\to\C\) be two continuous differentiable paths that intersect at \(0\text{:}\) \(\gamma_1(0) = \gamma_2(0)\text{.}\) Suppose also that \(\gamma_1'(0) \ne 0\) and \(\gamma_2'(0) \ne 0\text{.}\) Define the angle between \(\gamma_1\) and \(\gamma_2\) as follows.
    \begin{equation*} \angleop(\gamma_1,\gamma_2) = \arg(\gamma_1'(0)) - \arg(\gamma_2'(0)). \end{equation*}
    A function \(f\colon \C\to\C\) is said to be conformal at \(z_0\) if whenever \(\gamma_1,\gamma_2\colon[-1,1]\to\C\) are two continuously differentiable curves with \(\gamma_1(0) = \gamma_2(0) = z_0\) and \(\gamma_1'(0) \ne 0\) and \(\gamma_2'(0) \ne 0\text{,}\) then
    \begin{equation*} \angleop(\gamma_1,\gamma_2) = \angleop(f\circ \gamma_1, f\circ \gamma_2) \end{equation*}
    (and \((f\circ \gamma_1)'(0)\ne 0\) and \((f\circ \gamma_2)'(0) \ne 0\text{,}\) so that \(\angleop(f\circ \gamma_1, f\circ \gamma_2\) is defined).
    The point of all this is the following theorem.

    Theorem 3.5.2.

    If \(f\colon D\to \C\) is analytic at \(z_0 \in D\) and \(f'(z_0) \ne 0\text{,}\) then \(f\) is conformal at \(z_0\text{.}\)

    Proof.

    This is basically just the Chain Rule and the fact that \(\arg(z/w) = \arg(z) - \arg(w)\text{.}\)
    \begin{align*} \angleop( f\circ \gamma_1, f\circ \gamma_2) \amp \arg((f\circ\gamma_1)'(0)) - \arg((f\circ\gamma_2)'(0))\\ \amp \arg(f'(z_0) \gamma_1'(0)) - \arg(f'(z_0)\gamma_2'(0))\\ \amp \arg\left( \frac{f'(z_0) \gamma_1'(0)}{f'(z_0)\gamma_2'(0)}\right)\\ \amp \arg(\gamma_1'(0)) - \arg(\gamma_2'(0))\\ \amp \angleop(\gamma_1,\gamma_2) \end{align*}
    That's it!
    There is a converse to this theorem that we state for completeness but we won't prove.

    Fact 3.5.3.

    Suppose that \(f\) is continuously differentiable and conformal on a domain \(D\) with nonzero gradient at every point in \(D\text{.}\) Then \(f\) is analytic on \(D\text{,}\) and \(f'(z) \ne 0\) for every \(z\in D\text{.}\)

    Example 3.5.4.

    The squaring function \(f(z) = z^2\) is (as a polynomial) analytic everywhere. It maps the sector \(\arg z \in (-\theta_0,\theta_0)\) to the sector \(\arg z \in (-2\theta_0,2\theta_0)\) of twice the aperture. Notice that this function is not conformal at \(0\) since its derivative is zero there!

    Example 3.5.7.

    The exponential function \(z\mapsto e^z\) is conformal at every \(z\in\C\text{.}\) It restricts to a conformal equivalence between the horizontal strip \(\Im z \in (-\pi,pi)\) to the slit plane \(\C\wo(-\infty,0]\text{.}\)

    Definition 3.5.9.

    If \(U\) and \(V\) are open subsets of \(\C\) and \(f\colon U\to V\) is a conformal bijection, then \(f\) is called a conformal equivalence.

    Investigation 3.5.1.

    Don't forget that conformality is equivalent to analytic with derivative \(0\text{.}\) So the Inverse Function Theorem implies that a conformal bijection has local conformal inverses, which can be patched together to give a global conformal inverse. All this to say that conformal equivalence is a symmetric relation: if \(U\) is conformally equivalent to \(V\text{,}\) then \(V\) is conformally equivalent to \(U\text{.}\)
    Motivation: Suppose that we want to solve 2D Laplace's Equation on a domain \(U\) subject to some boundary conditions. Maybe \(U\) isn't so nice. We might try to find a conformally equivalent domain \(V\text{,}\) solve Laplace's Equation on \(V\text{,}\) and then transfer the solution back to \(U\text{.}\)
    So it is useful to have a library of conformal equivalences.

    Example 3.5.10.

    The map \(z\mapsto z^n\text{,}\) \(n\ge 2\text{,}\) defines a conformal equivalence
    \begin{equation*} \set{ z\in \C}{0 \lt \arg(z) \lt \frac{\pi}{n}} \to \bH, \end{equation*}
    where \(\bH\) is the upper halfplane \(\Im z > 0\text{.}\)

    Example 3.5.12.

    The map \(z\mapsto \frac{z-i}{z+i}\) defines a conformal equivalence between \(\bH\) and the unit disk \(\set{z}{\abs{z} \lt 1}\text{.}\)
    (Why does this function even send the upper halfplane to the disk? Well, \(z\in \bH\) iff \(z\) is closer to \(i\) than to \(-i\text{,}\) iff \(\abs{z-i} \lt \abs{z+i}\text{,}\) iff \(\abs{\frac{z-i}{z+i}} \lt 1\text{.}\))
    Why is it conformal? Why is it a bijection? For the answers to those and other questions, look to the next Section!!!
