Definition 5.2.13.

A isolated singularity of \(f\) at \(z_0\) is ...
  • removable if \(f\) is bounded near \(z_0\) (as in Proposition 5.2.11);
  • a pole (of order \(k\)) if it satisfies the hypotheses of Proposition 5.2.12;
  • an isolated essential singularity if it is an isolated singularity that is neither removable nor a pole.
