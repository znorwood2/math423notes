In summary, if \(f\) has no essential singularities inside \(\gamma\text{,}\) then
\begin{align*} \oint_\gamma \frac{f'(z)}{f(z)}\,dz \amp = 2\pi i \left[ (\text{sum of zeros of } \text{ inside } ) - (\text{sum of poles of } \text { inside } ) \right] \\ \amp = 2\pi i (\text{number of zeros inside } - \text{ number of poles inside } ) \end{align*}
where the sums count multiplicity.
