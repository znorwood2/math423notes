Suppose that \(R\) is the region bounded between two closed curves \(\gamma_1\) and \(\gamma_2\text{,}\) each oriented counterclockwise. (E.g. \(R\) could be an annulus. See Figure 4.3.10.) We will show that if \(f\) is analytic on \(R\) then
\begin{equation*} \int_{\gamma_1} f(z)\,dz = \int_{\gamma_2} f(z)\,dz. \end{equation*}
Cut the region by \(\gamma_3\text{,}\) as shown in the figure. The slit region enclosed by the curve \(\gamma_1 + \gamma_3 - \gamma_2 - \gamma_3\) is simply connected, so we can apply Cauchy's Theorem to conclude that
\begin{equation*} \int_{\gamma_1 + \gamma_3 - \gamma_2 - \gamma_3} f(z)\,dz = 0. \end{equation*}
But we decompose this using Lemma 4.2.2:
