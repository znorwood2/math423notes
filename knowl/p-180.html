Consider the conjugation map \(f(z) = \bar z\text{.}\) Then \(f = u + iv\) with \(u(x,y) = x\) and \(v(x,y) = -y\text{.}\) In this case we have \(u_x = 1\) \(v_y = -1\text{,}\) so the Cauchy–Riemann Equations hold nowhere. So the conjugation map is nowhere (complex-)differentiable, which might seem weird since it is a nice orthogonal transformation. In Section 3.5 we will get another explanation: conjugation is not analytic because it fails to preserve angles, in particular their orientation.
