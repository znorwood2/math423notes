A set \(U\subseteq \C\) is path-connected iff any two points in \(U\) can be joined by a continuous path that does not leave \(U\text{;}\) that is, iff for all \(z,w\in U\) there is a continuous function \(\gamma\colon[0,1]\to U\) with \(\gamma(0) = z\) and \(\gamma(1) = w\text{.}\)
