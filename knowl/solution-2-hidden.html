First notice that \(\frac{\partial^2}{\partial x^2} xy = 0 = -\frac{\partial^2}{\partial y^2} xy\text{,}\) so \(u\) is harmonic.
Solve the CR equations. Starting with
\begin{equation*} y = u_x = v_y \end{equation*}
we conclude that \(v(x,y) = \frac{y^2}{2} + h(x)\) for some function \(h\text{.}\) Differentiating wrt \(x\) and using the CR equations again gives
\begin{equation*} x = u_y = -v_x = h'(x), \end{equation*}
which means (calculus) that \(h(x) = -\frac{x^2}{2} + C\text{.}\) We conclude that \(v(x,y) = \frac{y^2}{2} - \frac{x^2}{2} + C\text{.}\)
What analytic function is \(u+iv\text{?}\) Look:
\begin{equation*} f(z) = f(x+iy) = xy + \left( \frac{y^2}{2} - \frac{x^2}{2} + C \right)i = -i\frac{z^2}{2} + C. \end{equation*}
