Proof

The key is (4.4.1), which decomposes \(\int_\gamma f(z)\,dz\) into its real and imaginary parts. Each of those parts is a vector integral to which we can apply Green's Theorem. Writing \(R\) for the region enclosed by \(\gamma\text{,}\) we have the following.
\begin{align*} \text{Real part:}\quad \int_\gamma u\,dx - v\,dy \amp = \iint_R (- v_x - u_y)\,dxdy = \iint 0 = 0 \\ \text{Imaginary part:}\quad \int_\gamma v\,dx + u\,dy \amp = \iint_R (u_x - v_y)\,dxdy = \iint 0 = 0 \end{align*}
