  3. Write the Laurent series for \(f\) centered at \(a\) and take the limit term by term.
  4. Since \(f\) has a simple pole at \(a\) the Laurent series of \(f\) near \(a\) can be written as
    \begin{equation*} f(z) = \frac{c_{-1}}{z-a} + c_0 + c_1(z-a) + \cdots \end{equation*}
    and \(g\) has a Taylor series given as follows.
    \begin{equation*} g(z) = d_0 + d_1(z-a) + d_2(z-a)^2 + \cdots \end{equation*}
    Multiply these together:
    \begin{equation*} f(z)g(z) = \frac{c_{-1}d_0}{z-a} + (c_0 d_0 + c_{-1} d_1) + \cdots \end{equation*}
    Read off the residue from this:
    \begin{equation*} \Res\limits_{z=a} f(z) g(z) = d_0 c_{-1} = g(a) \Res\limits_{z=a}f(z). \end{equation*}
    The second statement follows from the first since \(1/g\) is analytic at \(a\text{.}\)
  5. Since \(g\) has a simple zero at \(a\text{,}\) its Taylor series around \(a\) looks like this:
    \begin{equation*} g(z) = c_1 (z-a) + c_2 (z-a)^2 + \cdots \end{equation*}
    with \(c_1 \ne 0\text{.}\) Use this to express \(\frac{1}{g(z)}\) as follows.
    \begin{equation*} \frac{1}{g(z)} = \frac{1}{c_1 (z-a) + c_2 (z-a)^2 + \cdots} = \frac{1}{c_1(z-a)}\left( \frac{1}{1 + \frac{c_2}{c_1}(z-a) + \cdots} \right) \end{equation*}
    The function \(\frac{1}{1 + \frac{c_2}{c_1}(z-a) + \cdots}\) is analytic at \(z=a\) and takes the value \(1\) at \(a\text{.}\) Thus it has a Taylor expansion \(1 + d_1(z-a)+ \cdots\text{.}\) Combine this to see that the Laurent expansion of \(1/g\) near \(a\) takes the following form.
    \begin{equation*} \frac{1}{c_1 (z-a)} (1 + d_1 (z-a) + \cdots) \end{equation*}
    So the residue of \(1/g\) at \(a\) is \(\frac{1}{c_1} = \frac{1}{g'(a)}\text{.}\)
