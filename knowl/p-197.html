Don't forget that conformality is equivalent to analytic with derivative \(0\text{.}\) So the Inverse Function Theorem implies that a conformal bijection has local conformal inverses, which can be patched together to give a global conformal inverse. All this to say that conformal equivalence is a symmetric relation: if \(U\) is conformally equivalent to \(V\text{,}\) then \(V\) is conformally equivalent to \(U\text{.}\)
