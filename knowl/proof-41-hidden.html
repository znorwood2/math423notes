Look at the function \(h(z) = f(z) - g(z)\) on \(D\text{.}\) Then \(h\) has a nonisolated zero on \(D\text{,}\) so there is a small disk \(B_\eps(w)\) on which \(h\) is identically zero.
Now we need a topological argument. Look at the following two sets.
\begin{align*} U \amp = \set{ a \in D}{h \equiv 0 \text{ on some neighborhood } B_\eps(a) \text{ of } a \text{ in } D} \\ V \amp = \set{ a \in D}{ \text{there is } n \ge 0 \text{ for which } h^{(n)}(a) \ne 0 } \end{align*}
Certainly \(a \in U\) implies \(a \notin V\text{;}\) i.e., \(U \cap V \ne \emptyset\text{.}\) And \(U \cup V = D\) by Taylor's Theorem: expand around \(a\) and look at the Taylor coefficients. But \(D\) is (path-)connected, so either \(U = \emptyset\) or \(V = \emptyset\text{.}\) We have assumed that \(w \in U\text{,}\) so \(V\) must be empty. Thus \(h\) is identically zero on all of \(D\text{.}\)
