Fix \(\eps \lt r\text{.}\) Use the Mean Value Property and the fact that \(\abs{f}\) achieves its maximum at \(z_0\text{:}\)
\begin{align*} \abs{f(z_0)} \amp = \abs{ \frac{1}{2\pi} \int_0^{2\pi} f(z_0 + \eps e^{i\theta})\,d\theta } \\ \amp \le \frac{1}{2\pi} \int_0^{2\pi} \abs{f(z_0 + \eps e^{i\theta})}\,d\theta \\ \amp \le \frac{1}{2\pi} \int_0^{2\pi} \abs{f(z_0)}\,d\theta \\ \amp = \frac{1}{2\pi} 2\pi \abs{f(z_0)} \end{align*}
So \(\abs{f(z_0)} = \frac{1}{2\pi} \int_0^{2\pi} \abs{f(z_0 + \eps e^{i\theta})}\,d\theta\text{,}\) i.e.,
\begin{equation*} \int_0^{2\pi} \abs{f(z_0+\eps e^{i\theta})} - \abs{f(z_0)}\,d\theta = 0. \end{equation*}
The integrand \(\abs{f(z_0+\eps e^{i\theta})} - \abs{f(z_0)}\) is a continuous nonnegative real-valued function on \([0,2\pi]\) that integrates to \(0\text{.}\) Thus this function must be \(0\text{,}\) by a standard argument in real analysis. So \(\abs{f(z_0)} = \abs{f(z_0 + \eps e^{i\theta})}\) for all \(\theta \in [0,2\pi]\text{.}\)
