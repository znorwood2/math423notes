Next we deal with the integral over \(A_R\) using an ML estimate. For \(z\) in the upper halfplane,
\begin{equation*} \abs{e^{iaz}} = e^{ia(x + iy)} = e^{ay} \le 1, \end{equation*}
since \(a \gt 0\text{.}\) So the ML estimate can be done almost exactly as in the previous example:
\begin{equation*} \abs{\int_{A_R} \frac{e^{iaz}}{1 + z^2}\,dz} \le \frac{1}{R^2 - 1}\pi R \underset{R\to\infty}{\longrightarrow} 0. \end{equation*}
