    Section 4.2 Basic properties and the Fundamental Theorem

    Lemma 4.2.1.

    The integral \(\int_\gamma f(z)\, dz\) is independent of parametrization. That is, if \(\gamma\colon [a,b]\to\C\text{,}\) \(\delta\colon [a',b']\to\C\text{,}\) and \(\phi\colon [a',b']\to[a,b]\) satisfy \(\delta = \gamma \circ \phi\text{,}\) then
    \begin{equation*} \int_\gamma f(z)\,dz = \int_\delta f(z)\,dz. \end{equation*}

    Proof.

    This follows from the definition of the contour integral and the substitution rule (aka ‚Äúu-sub‚Äù) for the usual Calc I integral:
    \begin{align*} \int_\delta f(z)\,dz \amp = \int_{a'}^{b'} f(\gamma(\phi(t)))(\gamma\circ\phi)'(t)\,dt\\ \amp = \int_{a'}^{b'} f(\gamma(\phi(t))) \gamma'(\phi(t)) \phi'(t)\,dt \\ \amp = \int_a^b f(\gamma(u)) \gamma'(u) \,du \\ \amp = \int_\gamma f(z)\,dz \end{align*}
    (For the third equality, we did a u-substitution with \(u = \gamma(\phi(t))\text{.}\))

    Lemma 4.2.2.

    1. \(\int_\gamma f(z) + g(z)\,dz = \int_\gamma f(z)\,dz + \int_\gamma g(z)\,dz\) and \(\int_\gamma cf(z)\,dz = c \int_\gamma f(z)\,dz\text{.}\)
    2. Suppose that \(\gamma\colon[a,b]\to\C\) is a contour and that \(c\in [a,b]\text{.}\) Let \(\gamma_{\text{start}}\) be the restriction of \(\gamma\) to \([a,c]\) and let \(\gamma_{\text{end}}\) be the restriction of \(\gamma\) to \([c,b]\text{.}\) Then
      \begin{equation*} \int_\gamma f(z)\,dz = \int_{\gamma_{\text{start}}} f(z)\,dz + \int_{\gamma_{\text{end}}} f(z)\,dz \end{equation*}
    3. \begin{equation*} \int_{-\gamma} f(z)\,dz = -\int_\gamma f(z)\,dz \end{equation*}
      where \(-\gamma\colon [a,b] \to \C\) is the reverse of \(\gamma\) defined by \((-\gamma)(t) = a+b - t \text{.}\)
    4. \begin{equation*} \abs{\int_\gamma f(z)\,dz} \le \operatorname{length}(\gamma) \cdot \sup \abs{f(\gamma(t))}, \end{equation*}
      where \(\operatorname{length}(\gamma) = \int_a^b \abs{\gamma'(t)}\,dt\text{.}\) (This is the same definition of arclength from multivariable calculus.)

    Proof.

    We'll prove Item¬†d, but the proofs of the remaining properties are left as exercises.
    Property Item¬†d follows from the definition and the similar ‚Äúbounding box property‚Äù of standard Calc I integrals:
    \begin{align*} \abs{\int_\gamma f(z)\,dz} \amp = \abs{\int_a^b f(\gamma(t))\gamma'(t)\,dt} \\ \amp \le \int_a^b \abs{f(\gamma(t))} \abs{\gamma'(t)}\,dt \\ \amp \le \left( \sup_{t\in [a,b]} \abs{f(\gamma(t))} \right) \int_a^b \abs{\gamma'(t)}\,dt \end{align*}
    That's what we wanted to prove.

    Example 4.2.3.

    Take the standard parametrization of the unit circle: \(\gamma\colon [0,2\pi]\to\C\text{,}\) \(\gamma(\theta) = e^{i\theta}\text{.}\) Let \(f(z) = z^n\) for an unspecified \(n\in\mathbb{Z}\text{.}\) Let's compute \(\int_\gamma f(z)\,dz\text{:}\)
    \begin{equation*} \int_\gamma f(z)\,dz = \int_0^{2pi} (e^{i\theta})^n \cdot i e^{i\theta} \, d\theta = i\int_0^{2\pi} e^{i(n+1)\theta}\,d\theta, \end{equation*}
    which equals \(2\pi i\) if \(n = -1\) and \(0\) otherwise.
    We aim to prove a version of the Fundamental Theorem of Calculus for contour integrals. First we need a simple version for functions \(g\colon [a,b] \to \C\text{:}\)

    Lemma 4.2.4.

    Suppose that \(g(t) = u(t) + iv(t)\) and \(G(t) = U(t) + iV(t)\) are two paths \([a,b]\to\C\) for which \(G'(t) = g(t)\text{.}\) Then \(\int_a^b g(t)\,dt = G(b) - G(a)\text{.}\)

    Proof.

    This follows immediately from the defintion of integrals for paths like this and the Fundamental Theorem of Calculus.
    \begin{equation*} \int_a^b g(t)\,dt = \int_a^b u(t)\,dt + i\int_a^b v(t)\,dt = U(b) - U(a) + i(V(b) - V(a)) = G(b) - G(a). \end{equation*}

    Theorem 4.2.5.

    If \(F'(z) = f(z)\) on a domain \(U\) and \(\gamma \colon [a,b]\to U\) is a contour, then \(\int_\gamma f(z)\,dz = F(\gamma(b)) - F(\gamma(a))\text{.}\)

    Proof.

    Apply Lemma¬†4.2.4 to \(G = F\circ \gamma\) and \(g = f\text{.}\)

    Example 4.2.6.

    Let's revisit Example¬†4.2.3. We're considering \(f(z) = z^n\) and \(\gamma(t) = e^{it}\) for \(t \in [0,2\pi]\text{.}\) If \(n\ne -1\text{,}\) then \(F(z) = \frac{z^{n+1}}{n+1}\) is an antiderivative of \(f\) on \(\C\text{.}\) So \(\int_\gamma z^n \, dz = F(0) - F(0) = 0\text{.}\)
    On the other hand, if \(n = -1\text{,}\) then \(f(z) = \frac{d}{dz} \log(z)\text{,}\) but the complex logarithm can be defined continuously only on a slit plane. In fact, because \(\int \frac{1}{z} \, dz\) does not vanish on the unit circle, there cannot be a continuous branch of the log defined on any domain \(U\) that includes the unit circle. Now we have a proof of that fact!
    An especially important consequence of the FTC for contour integrals is that if \(f\) has an antiderivative, then \(f\) integrates to \(0\) over any closed curve. As we will see in the next section, this implication reverses!
