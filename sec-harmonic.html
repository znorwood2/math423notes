    Section 3.4 Harmonic functions

    Motivated by Laplace's Equation
    \begin{equation*} \Delta u = \frac{\partial^2 u}{\partial x_1^2} + \cdots + \frac{\partial^2 u}{\partial x_n^2} = 0, \end{equation*}
    which is very important in mathematical physics, we make the following definition.

    Definition 3.4.1.

    A function \(u = u(x,y)\colon \R^2 \to \R\) is harmonic if the partial derivatives \(u_x, u_y,u_{xx}, u_{xy}, u_{yx}, u_{yy}\) all exist and are continuous, and \(\Delta u = 0\text{,}\) i.e., \(u_{xx} + u_{yy} = 0\text{.}\)

    Theorem 3.4.2.

    If \(u+iv\) is analytic, then \(u\) and \(v\) are harmonic functions.

    Proof.

    Later we will show that the real and imaginary parts of analytic functions have continuous partial derivatives of all orders. So let's not worry about that part for now.
    This is just an application of the Cauchy‚ÄìRiemann Equations and equality of mixed partials (sometimes called Clairaut's Theorem by calculus textbooks):
    \begin{align*} u_{xx} \amp = \frac{\partial}{\partial x} u_x \amp\amp \\ \amp = \frac{\partial}{\partial x} v_y \amp\amp \text{(CR)} \\ \amp = \frac{\partial}{\partial y} v_x \amp\amp \text{(Clairaut)} \\ \amp = \frac{\partial}{\partial y} (-u_y) \amp\amp \text{(CR)} \\ \amp = -u_{yy} \end{align*}
    Rearranging gives \(u_{xx} + u_{yy} = 0\text{.}\) A very similar proof works for \(v\text{.}\)

    Example 3.4.3.

    The function \(u(x,y) = e^x \cos(y)\) is harmonic, since it is the real part of the analytic function \(f(z) = e^z\text{.}\)

    Definition 3.4.4.

    If \(u\) is harmonic on a domain \(D\) and \(v\) is a function for which \(u + iv\) is analytic on \(D\text{,}\) then we say that \(v\) is a harmonic conjugate of \(u\text{.}\)

    Checkpoint 3.4.5.

    State and prove something to the effect that harmonic conjugates are unique up to an additive constant.
    Solution.
    Suppose that \(u+iv\) and \(u + iw\) are both analytic. Then \(\frac{i(v - w)}{2}\) is analytic too, but it is also real-valued. So \(v - w\) is constant.

    Example 3.4.6.

    Find a harmonic conjugate for \(u(x,y) = xy\text{.}\)
    Solution.
    First notice that \(\frac{\partial^2}{\partial x^2} xy = 0 = -\frac{\partial^2}{\partial y^2} xy\text{,}\) so \(u\) is harmonic.
    Solve the CR equations. Starting with
    \begin{equation*} y = u_x = v_y \end{equation*}
    we conclude that \(v(x,y) = \frac{y^2}{2} + h(x)\) for some function \(h\text{.}\) Differentiating wrt \(x\) and using the CR equations again gives
    \begin{equation*} x = u_y = -v_x = h'(x), \end{equation*}
    which means (calculus) that \(h(x) = -\frac{x^2}{2} + C\text{.}\) We conclude that \(v(x,y) = \frac{y^2}{2} - \frac{x^2}{2} + C\text{.}\)
    What analytic function is \(u+iv\text{?}\) Look:
    \begin{equation*} f(z) = f(x+iy) = xy + \left( \frac{y^2}{2} - \frac{x^2}{2} + C \right)i = -i\frac{z^2}{2} + C. \end{equation*}
    An abstract version of the method used in Example¬†3.4.6 can be used to prove the following general theorem.

    Theorem 3.4.7.

    Let \(D\) be an open disk or an open (filled-in) rectangle. Let \(u = u(x,y)\) be a harmonic function on \(D\text{.}\) Then \(u\) has a harmonic conjugate \(v\) on \(D\text{,}\) unique up to adding a constant.
    The ‚Äúright‚Äù class of domains \(D\) to which Theorem¬†3.4.7 applies is the class of simply connected domains. A domain is simply connected if and only if every loop in \(D\) can be shrunk continuously to a point without leaving \(D\text{.}\) E.g. a disk is simply connected but an annulus is not.
    NB. Theorem¬†3.4.7 does not hold in general for domains that are not simply connected, e.g. \(D\) an annulus or \(D = \C \wo\{0\}\text{.}\) For example, consider \(u(x,y) = \log\sqrt{x^2 + y^2}\text{.}\) You will probably be asked on the homework to explore this example.
