    Section 3.2 The Cauchy‚ÄìRiemann Equations

    The limit of the difference quotient from the definition of derivative
    \begin{equation*} \lim_{\Delta z \to 0} \frac{f(z + \Delta z) - f(z)}{\Delta z} \end{equation*}
    is really a multivariable calculus limit:
    \begin{equation*} \lim_{(\Delta x, \Delta y) \to (0,0)} \frac{f((x + \Delta x) + i(y + \Delta y)) - f(x + iy)}{\Delta x + i\Delta y} \end{equation*}
    If this limit exists (meaning that \(f = u + iv\) is differentiable at \(z\)), then it must exist along the two coordinate directions. That is, in the special cases \(\Delta z = \Delta x\) and \(\Delta z = i\Delta y\) we must get the same result. Let's explore that.
    \begin{align} \frac{f(z + \Delta x) - f(z)}{\Delta x} \amp = \frac{u(z + \Delta x,y) + iv(x + \Delta x,y) - u(x,y) - iv(x,y)}{\Delta x} \notag\\ \amp = \frac{u(x + \Delta x,y) - u(x,y)}{\Delta x} + i \frac{v(x + \Delta x,y) - v(x,y)}{\Delta x} \tag{3.2.1} \end{align}
    If \(f'(z)\) exists, then as \(\Delta x\) tends to \(0\) the expression in (3.2.1) must tend toward \(f'(z)\text{.}\) But it must also equal \(\frac{\partial u}{\partial x}(x,y) + i \frac{\partial v}{\partial x}(x,y)\text{,}\) by the definition of partial derivatives! In other words, for \(f = u + iv\) and \(z = x + iy\text{,}\)
    \begin{equation*} f'(z) = \frac{\partial u}{\partial x}(x,y) + i \frac{\partial v}{\partial x} (x,y). \end{equation*}
    Do the same thing in the \(y\)-direction with \(\Delta z = i \Delta y\) and you'll get the following.
    \begin{align*} \frac{f(z + i\Delta y) - f(z)}{i \Delta y} \amp = \frac{v(x,y + \Delta y) - v(x,y)}{\Delta y} - i \frac{u(x,y + \Delta y) - u(x,y)}{\Delta y} \\ \amp \underset{\Delta y \to 0}{\longrightarrow} \frac{\partial v}{\partial y} (x,y) - i \frac{\partial u}{\partial y}(x,y) \end{align*}
    More compactly, \(f'(z) = u_x + iv_x = v_y - iu_y\text{.}\) Identifying real and imaginary parts, we obtain the Cauchy‚ÄìRiemann Equations.

    Theorem 3.2.1. Cauchy‚ÄìRiemann Equations.

    Suppose that \(f = u + iv\) is defined on a domain \(U\subseteq \C\) and that \(z = x + iy \in U\text{.}\) Then \(f\) is (complex-)differentiable at \(z\) if and only if \(u\) and \(v\) (construed as functions \(\R^2 \to \R\)) have continuous partial derivatives that satisfy the following equations, called the Cauchy‚ÄìRiemann Equations, at \(z\text{.}\)
    \begin{align*} u_x \amp = v_y \\ u_y \amp = -v_x \end{align*}
    In that case, we have
    \begin{equation*} f'(z) = u_x(x,y) + i v_x(x,y) = v_y(x,y) - i u_y(x,y). \end{equation*}
    We have basically shown how to prove the forward direction. We will not give the proof of the other direction; it is an exercise in multivariable real analysis using Taylor's Theorem.

    Example 3.2.2.

    1. Let \(f(z) = z^2\text{,}\) so that \(f = u + iv\) where \(u(x,y) = x^2 - y^2\) and \(v(x,y) = 2xy\text{.}\) We know that this function (a polynomial) is differentiable everywhere, so we expect \(u\) and \(v\) to satisfy the Cauchy‚ÄìRiemann Equations everywhere. Indeed, they do:
      \begin{align*} u_x \amp = 2x = v_y \\ u_y \amp = -2y = -v_x \end{align*}
    2. Consider the conjugation map \(f(z) = \bar z\text{.}\) Then \(f = u + iv\) with \(u(x,y) = x\) and \(v(x,y) = -y\text{.}\) In this case we have \(u_x = 1\) \(v_y = -1\text{,}\) so the Cauchy‚ÄìRiemann Equations hold nowhere. So the conjugation map is nowhere (complex-)differentiable, which might seem weird since it is a nice orthogonal transformation. In Section¬†3.5 we will get another explanation: conjugation is not analytic because it fails to preserve angles, in particular their orientation.
    3. The exponential map \(f(z) = e^z\) has real part \(u(x,y) = e^x \cos y\) and imaginary part \(v(x,y) = e^x \sin y\text{.}\) We see that the Cauchy‚ÄìRiemann equations
      \begin{align*} u_x \amp = e^x \cos y = v_y \\ u_y \amp = -e^x \sin y = -v_x \end{align*}
      hold everywhere, as expected since \(f\) is entire. We also observe that
      \begin{equation*} f'(z) = u_x + i v_x = e^x \cos y + i e^x \sin y = e^z. \end{equation*}
    Using the Cauchy‚ÄìRiemann Equations, we will several rigidity properties of analytic functions that start to give us a sense of what sort of functions can be analytic. We begin with an unsurprising complex analogue of a standard fact from single-variable calculus.

    Lemma 3.2.3.

    If \(f\) is analytic on a domain \(D\) and \(f'(z) = 0\) for every \(z\in D\text{,}\) then \(f\) must be constant on \(D\text{.}\)

    Proof.

    Write \(f = u+iv\text{,}\) so that our assumption and the Cauchy‚ÄìRiemann Equations give
    \begin{equation*} 0 = f'(z) = u_x + iv_x = v_y - i u_y. \end{equation*}
    Identifying real and imaginary parts, we see that \(u_x = u_y = v_x = v_y = 0\) on \(D\text{.}\) It follows that the directional derivative \(D_{\mathbf{w}}f = \grad w \cdot \mathbf{w} = 0\) for every direction vector \(\mathbf{w}\text{,}\) at every point of \(D\text{.}\) Using this, one can show (using the fact that a plain ole function \(\R\to\R\) with derivative \(0\) must be constant) that \(f\) is constant along any line segment that lies completely in \(D\text{.}\) But, since \(D\) is path-connected, any two points \(z,w\in D\) can be joined by a finite sequence of line segments.
    This actually requires a togological argument that we've swept under the rug. The gist of it is this: every point on a path from \(z\) to \(w\) has an open neighborhood that is included in \(D\text{,}\) since \(D\) is open. Only finitely many of these open neighborhoods are needed to cover the path by compactness. Now replace the portion of the path in the first ball with the straight-line path to where it exits the ball. Repeat with the remaining balls.
    Since \(f\) is constant along each of these line segments, \(f(z) = f(w)\text{.}\) So \(f\) is constant on \(D\text{.}\)

    Checkpoint 3.2.4.

    If \(f\) is analytic and real-valued on a domain \(D\text{,}\) then \(f\) must be constant on \(D\text{.}\)

    Proposition 3.2.5.

    If \(f\) and \(\bar f\) are both analytic on a domain \(D\text{,}\) then \(f\) is constant on \(D\text{.}\)

    Proof.

    There is a short proof from Checkpoint¬†3.2.4, but we will give a direct proof that does not appeal to a homework problem.
    Write \(f = u+iv\) so that \(\bar f = u - iv\text{.}\) The Cauchy‚ÄìRiemann Equations for \(f\) say:
    \begin{equation*} u_x = v_y \quad \text{and} \quad u_y = -v_x, \end{equation*}
    while the Cauchy‚ÄìRiemann Equations for \(\bar f\) say:
    \begin{equation*} u_x = -v_y \quad \text{and} \quad u_y = v_x. \end{equation*}
    Adding these equations together gives \(u_x = u_y = v_x = v_y = 0\text{.}\) So \(f'(z) = u_x + i u_y = 0\) on \(D\text{.}\) By Lemma¬†3.2.3, it follows that \(f\) is constant on \(D\text{.}\)

    Corollary 3.2.6.

    If \(f\) is analytic on a domain \(D\) and \(\abs{f(z)}\) is constant on \(D\text{,}\) then \(f\) is constant on \(D\text{.}\)

    Proof.

    If \(\abs{f}\) takes the constant value \(0\) on \(D\text{,}\) then \(f\) takes the constant value \(0\) on \(D\text{.}\) So assume from now on that the constant value of \(\abs{f}\) on \(D\) is not zero. Recall that \(\abs{f(z)} = f(z)\bar{f(z)}\text{.}\) Rearrange to see that \(\bar{f(z)} = \frac{1}{\abs{f(z)}^2} f(z)\text{,}\) which means that \(\bar f\) is a constant multiple of the analytic function \(f\text{;}\) therefore \(\bar f\) is analytic. Now we can apply Proposition¬†3.2.5 to conclude that \(f\) is constant on \(D\text{.}\)
