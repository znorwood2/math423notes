Example 1.4.21.

Let's find all possible values of \(i^i\text{.}\)
\begin{equation*} i^i = e^{i \log i} = e^{i(\ln 1 + i \arg i)} = e^{-\arg i}. \end{equation*}
Since the possible values of \(\arg i\) are \(\frac{\pi}{2} + 2\pi m\text{,}\) \(m\in\mathbb{Z}\text{,}\) we have
\begin{equation*} i^i = e^{-\pi/2} = e^{-5\pi / 2} = \cdots \end{equation*}
These are all real numbers!!!
