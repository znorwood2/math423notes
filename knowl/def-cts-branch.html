Definition 1.4.16.

If \(U\subseteq\C\wo\{0\}\) is an open set, then a branch of the log defined on \(U\) is a continuous function \(f\colon U\to\C\) for which \(e^{f(z)} = z\) for all \(z\in U\text{.}\)
A branch point of a multi-valued function \(f\) is a point \(p\in\C\) such that, for every \(\epsilon \gt 0\text{,}\) the function cannot be given a continuous single-valued definition on \(B_\epsilon(p) \wo \{p\}\text{.}\)
