Where does the Inverse Function Theorem come in? you might wonder. Consider a function \(f = u + iv\) that is analytic on a domain \(D\text{.}\) If we think of \(f\) as the function \((u,v) \colon \R^2 \to \R^2\text{,}\) then it has a Jacobian matrix
\begin{equation*} J_f = \begin {bmatrix} \frac{\partial u}{\partial x} \amp \frac{\partial u}{\partial y} \\ \frac{\partial v}{\partial x} \amp \frac{\partial v}{\partial y} \end {bmatrix} \end{equation*}
whose determinant can be computed using the Cauchy–Riemann Equations:
\begin{equation*} \det J_f = u_x v_y - u_y v_x = u_x^2 + v_x^2 = \abs{u_x + i v_x}^2 = \abs{f'}^2. \end{equation*}
So the Jacobian is zero exactly where the complex derivative is zero. This allows us to state Theorem 3.3.2 in terms of the complex derivative instead of referring to the Jacobian.
