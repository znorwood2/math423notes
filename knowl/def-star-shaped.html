Definition 2.1.7.

If \(X\subseteq \C\) and \(z_0\in X\text{,}\) then \(X\) is star-shaped with respect to \(z_0\) iff for all \(z\in X\) the straight line segment connecting \(z_0\) to \(z\) lies completely in \(X\text{.}\)
You could state this precisely by using the standard parametrization of the line segment. '\(X\) is star-shaped wrt \(z_0\) iff for all \(t\in [0,1]\) we have \((1-t)z_0 + tz \in X\text{.}\)'
A set \(X\) is convex if \(X\) is star-shaped with respect to every one of its elements. We just say that \(X\) is star-shaped if there is at least one \(p \in X\) with respect to which \(X\) is star-shaped.
