So, in polar form, \(a+i\) can be written as \(re^{i(\theta + 2k\pi)}\) for any \(k\in\mathbb{Z}\text{.}\) So we should be able to take \(z_0 = (re^{i\theta})^{1/2} = r^{1/2}e^{i\theta/2}\) and \(z_1 = -z_0\) as our two square-roots. But we still want them in rectangular coordinates, so we use Euler's Formula, some trig identities, and the fact that \(\cos\theta = a/r\text{.}\)
