Define \(h \colon D\to \C\) as follows.
\begin{equation*} h(z) = \begin {cases} (z- z_0)^2 f(z) \amp \text{if } z \ne z_0 \\ 0 \amp \text{if } z = z_0 \end {cases} \end{equation*}
Our assumption is that there are \(M\) and \(\eps\) such that \(\abs{f(z)} \lt M\) for all \(z \in B_\eps(z_0) \wo \{z_0\}\text{.}\) Let's estimate the difference quotient of \(h\) near \(z_0\text{:}\)
\begin{equation*} \abs{ \frac{h(z) - h(z_0)}{z - z_0}} = \frac{\abs{(z-z_0)^2 f(z)}}{\abs{z - z_0}} = \abs{z - z_0} \abs{f(z)} \le \abs{z - z_0} M, \end{equation*}
which tends to \(0\) as \(z\) tends to \(z_0\text{.}\) So \(h\) is differentiable at \(z_0\text{,}\) in addition to being differentiable near \(z_0\text{.}\) So by TheoremÂ 5.1.2 \(h\) has a power series expansion in a neighborhood \(B_\rho(z_0)\) of \(z_0\text{:}\)
\begin{equation*} h(z) = \sum_{n\ge 0} a_n (z - z_0)^n = a_0 + a_1(z - z_0) + a_2(z-a_0)^2 + \cdots \end{equation*}
We must have \(a_0 = a_1 = 0\text{,}\) so we can define
\begin{equation*} g(z) = \sum_{n\ge 0} a_{n+2} (z-z_0)^n = a_2 + a_3 (z-z_0) + a_4 (z - z_0)^2 + \cdots \end{equation*}
We have \(g(z) = \frac{h(z)}{(z-z_0)^2} = f(z)\) for \(z \in B_\rho(z_0)\wo \{z_0\}\text{.}\) And \(g(z) \to a_2\) as \(z \to z_0\text{,}\) so \(f(z) \to a_2\) as \(z \to z_0\text{.}\)
