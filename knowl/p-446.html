The integrand \(\abs{f(z_0+\eps e^{i\theta})} - \abs{f(z_0)}\) is a continuous nonnegative real-valued function on \([0,2\pi]\) that integrates to \(0\text{.}\) Thus this function must be \(0\text{,}\) by a standard argument in real analysis. So \(\abs{f(z_0)} = \abs{f(z_0 + \eps e^{i\theta})}\) for all \(\theta \in [0,2\pi]\text{.}\)
