Example 5.1.9.

Compute the power series expansion of \(f(z) = \sin(z)\) centered around \(a = 0\text{.}\)
Method 1: Compute derivatives of \(\sin(z)\text{:}\)
\begin{equation*} f^{(n)}(0) = \frac{d^n \sin(z)}{dz^n} = \begin {cases} (-1)^m \amp \text{ if } n = 2m+1 \\ 0 \amp \text{ if }n \text{ is even } \end {cases} \end{equation*}
Then use the formula.
Method 2: Recall that \(\sin(z) = \frac{e^{iz} - e^{-iz}}{2i}\text{,}\) so we can use the series for \(e^z\text{:}\)
\begin{align*} \sin(z) \amp = \frac{1}{2i} \left[ \sum_{n\ge 0} \frac{(iz)^n}{n!} - \sum_{n\ge 0} \frac{(-iz)^n}{n!} \right] \\ \amp = \frac{1}{2i} \sum_{n\ge 0} (1 - (-1)^n) \frac{i^n z^n}{n!} \\ \amp = \sum_{k\ge 0} (-1)^k \frac{z^{2k+1}}{(2k+1)!} \end{align*}
This series converges for all \(z\text{.}\) (This follows either from the fact that \(\sin(z)\) is entire, or from the fact that we built the series from the Taylor series for \(e^z\text{,}\) which converges everywhere.)
