Let's try to count the zeros of \(f(z) = z^5 - z - 1\) inside the circle \(\abs{z} = 2\text{.}\) Parametrize \(\gamma\) as \(\gamma(\theta) = 2e^{i\theta}\text{,}\) \(\theta \in [0,2\pi]\text{.}\) Then \(f\) has no poles, so its number of zeros inside \(\gamma\) is the winding number of \(f(\gamma)\) around \(0\text{.}\)
\begin{equation*} f(\gamma) = \gamma_1: \quad 32e^{5i\theta} - 2e^{i\theta} - 1, \quad \theta \in [0,2\pi] \end{equation*}
Let's analyze the simpler \(\gamma_2(t) = 32e^{5i\theta}\text{,}\) \(\theta \in [0,2\pi]\text{.}\) It traverses the large circle \(\abs{z} = 32\) five times CCW, so its winding number around \(0\) is \(5\text{.}\) Let original curve \(\gamma_2\) is pretty close to the original one \(\gamma_1\text{:}\)
\begin{equation*} \abs{ \gamma_2(\theta) - \gamma_1(\theta)} = \abs{2e^{i\theta} + 1} \le 3 \gg 32. \end{equation*}
We expect \(\gamma_1\) and \(\gamma_2\) to have the same winding number around \(0\text{...}\)
