A path is a continuous function \(\gamma\colon [a,b]\to\C\) for some \(a,b\in\R\text{.}\) A path \(\gamma\) is simple if it doesn't cross itself, except possibly at its endpoints. That is, \(\gamma\colon[a,b]\to\C\) is simple if whenever \(\gamma(t_1) = \gamma(t_2)\) then either \(t_1 = t_2\) or \(\{t_1,t_2\} = \{a,b\}\text{.}\) A path \(\gamma\) is closed if \(\gamma(a) = \gamma(b)\text{.}\) A contour is a path that is piecewise \(C^1\), i.e., a path that can be divided into finitely many paths each of which is continuously differentiable.
