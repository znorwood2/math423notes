    Section 1.2 Polar representation and Euler's Formula

    As a point in the plane, a complex number has a polar representation:
    \begin{equation*} z = x+iy = r(\cos\theta + i\sin\theta) \end{equation*}
    In precalculus you would've written \((x,y) = (r\cos\theta,r\sin\theta)\text{.}\) This is the same thing, just written using \(x+iy\) instead of \((x,y)\text{.}\)
    Notice that \(r\) is simply the modulus of \(z\text{:}\) \(r = \abs{z}\text{.}\) The angle \(\theta\) between the positive real axis and the ray \(\overrightarrow{0z}\) is called the argument of \(z\) and is denoted \(\arg(z)\text{.}\) Notice that \(\arg(z)\) is a multivalued function! Adding \(2\pi\) to \(\theta\) does not change \(z\text{.}\)

    Definition 1.2.2.

    The principal argument of a complex number \(z\text{,}\) denoted \(\Arg(z)\text{,}\) is the value of \(\theta\) that satisfies \(-\pi \lt \theta \le \pi\text{.}\)
    Warning: Some authors use the convention that \(\Arg(z) \in [0,2\pi)\text{.}\) Use caution when consulting other sources!
    So, for \(z\in\C\wo\{0\}\text{,}\) the possible values of \(\arg(z)\) are \(\Arg(z) + 2\pi k\text{,}\) where \(k\) can be any integer.

    Example 1.2.3.

    \(\Arg(i) = \frac{\pi}{2}\text{,}\) \(\Arg(1-i) = -\frac{\pi}{4}\text{.}\)

    Identity 1.2.4. Euler's Formula.

    For now we take the following as a definition; later we will see why it is justified.
    \begin{equation*} e^{i\theta} = \cos\theta + i\sin\theta \quad (\theta\in\R) \end{equation*}
    Using Euler's Formula, we obtain the exponential form of the polar representation of \(z\text{:}\)
    \begin{equation*} z = re^{i\theta} = \abs{z}e^{i\arg(z)} \end{equation*}

    Example 1.2.5.

    • \(\displaystyle e^{i\pi} = \cos\pi + i\sin\pi = -1 + 0i = -1\)
    • \(\displaystyle e^{i\pi/2} = \cos(\pi/2) + i\sin(\pi/2) = 0 + i = i\)
    • \(\displaystyle e^{i\pi/4} = \frac{1}{\sqrt2} + i\frac{1}{\sqrt2} \)

    Checkpoint 1.2.6.

    Use trig identities to prove the following identities about \(e^{i\theta}\)
    1. \(\displaystyle \abs{e^{i\theta}} = 1\)
    2. \(\displaystyle \bar{e^{i\theta}} = e^{-i\theta}\)
    3. \(\displaystyle \frac{1}{e^{i\theta}} = e^{-i\theta}\)

    Observation 1.2.7.

    Let's try applying Euler's Formula to \(e^{i(\alpha+\beta)}\) and then separately to \(e^{i\alpha}e^{i\beta}\text{.}\)
    \begin{equation*} e^{i(\alpha+\beta)} = \cos(\alpha + \beta) + i\sin(\alpha+\beta) \end{equation*}
    \begin{equation*} e^{i\alpha}e^{i\beta} = (\cos(\alpha) + i\sin(\alpha))(\cos(\beta) + i\sin(\beta)) = (\cos\alpha\cos\beta - \sin\alpha\sin\beta) + i(\cos\alpha\sin\beta + \cos\beta\sin\alpha) \end{equation*}
    The results of these two activities ought to be the same, and indeed if we identify real and imaginary parts we obtain the familiar addition formulas for sine and cosine!
    \begin{align*} \cos(\alpha+\beta) \amp = \cos\alpha\cos\beta - \sin\alpha\sin\beta \\ \sin(\alpha+\beta) \amp = \cos\alpha\sin\beta + \cos\beta\sin\alpha \end{align*}
    This is a useful way to derive the addition formulas for sine and cosine without memorizing them. But the way we've presented things is a bit backward; in fact, it's the trig identities that allow us to conclude that \(e^{i(\alpha+\beta)} = e^{i\alpha}e^{i\beta}\text{.}\)

    Checkpoint 1.2.8.

    Use trig identities to prove the following faacts about the argument.
    1. \(\displaystyle \arg \bar z = -\arg z\)
    2. \(\displaystyle \arg (\frac{1}{z}) = -\arg z\)
    3. \(\displaystyle \arg(zw) = \arg(z) + \arg(w)\)
    With the help of the polar representation, we are in a position to understand exactly how complex multiplication works. Writing \(z = re^{i\theta}\) and \(w = se^{i\phi}\text{,}\) we have
    \begin{equation*} zw = re^{i\theta}se^{i\phi} = rse^{i(\theta+\phi)}. \end{equation*}
    So the product \(zw\) has modulus \(\abs{z}\abs{w}\) and argument \(\arg z + \arg w\text{.}\)

    Subsection 1.2.1 De Moivre's Theorem

    We can identify real and imaginary parts in the following equation to obtain some trig identities.
    \begin{equation*} \cos(n\theta) + i\sin(n\theta) = e^{in\theta} = (e^{i\theta})^n = (\cos\theta + i\sin\theta)^n. \end{equation*}
    Let's illustrate this by considering the example \(n = 3\text{.}\) Identifying real and imaginary parts of the equation
    \begin{equation*} (\cos\theta + i\sin\theta)^3 = \cos^3 \theta - 3\cos\theta\sin\theta + i(3\cos^2\theta\sin\theta - \sin^3\theta) \end{equation*}
    gives
    \begin{align*} \cos(3\theta) \amp \cos^3\theta - 3\cos\theta\sin^2\theta \\ \sin(3\theta) \amp 3\cos^2\theta\sin\theta - \sin^3\theta \end{align*}

    Subsection 1.2.2 Roots of unity

    It is a special case of the Fundamental Theorem of Algebra ([cross-reference to target(s) "theorem-fta" missing or not unique]) that every nonzero complex number \(w\) has exactly \(n\) \(n\)th roots, i.e., solutions \(z\) to \(z^n = w\text{.}\) The case \(w = 1\) is already interesting (and of particular importance), so let's start there.
    Finding the \(n\) solutions to \(z^n = 1\) is made easier by considering the various polar representations of \(1\text{:}\)
    \begin{align*} 1 \amp = \cdots = e^{-4\pi i} = e^{-2\pi i} = e^{0 i} = e^{2\pi i} = e^{4\pi i} = \cdots \\ \amp = e^{2\pi i k}, \quad k\in\mathbb{Z} \end{align*}
    A reasonable guess (which turns out to be correct!) is that we can just divide the exponent on the righthand side by \(n\) to obtain the possible values of \(1^{1/n}\text{.}\)

    Definition 1.2.10.

    For an integer \(n \ge 1\text{,}\) the \(n\)th roots of unity are the complex numbers \(\omega_0,\dots, \omega_{n-1}\text{,}\) where
    \begin{equation} \omega_k = e^{2\pi i k / n}.\tag{1.2.1} \end{equation}

    Checkpoint 1.2.11.

    Prove that the \(n\) numbers \(\omega_0,\dots,\omega_{n-1}\) listed in (1.2.1) are in fact \(n\) different numbers!
    Explain why they appear equally spaced around the unit circle \(\abs{z} = 1\text{.}\)

    Example 1.2.12.

    The \(4\)th roots of unity are:
    \begin{align*} \omega_0 \amp e^{2\pi i 0 / 4} = 1\\ \omega_1 \amp e^{2\pi i 1 / 4} = e^{\pi i / 2} = i\\ \omega_2 \amp e^{2\pi i 2 / 4} = e^{\pi i} = -1\\ \omega_3 \amp e^{2\pi i 3 / 4} = e^{(3\pi / 2) i} = -i \end{align*}
    Don't forget Euler's Formula! E.g.
    \begin{align*} e^{(3\pi/2)i} \amp = \cos(3\pi/2) + i \sin(3\pi/2) \\ \amp = 0 + i(-1) \\ \amp = -i \text{.} \end{align*}
    We can find the \(n\)th roots of any nonzero complex number \(w = re^{i\theta}\) in a similar way. Suppose that \(\theta\) is, say, the principal argument of \(w\text{,}\) and consider the next \(n-1\) many arguments:
    \begin{equation*} w = re^{i\theta} = re^{i(\theta+2\pi)} = re^{i(\theta+4\pi)} = \cdots = re^{i(\theta+2(n-1)\pi)}. \end{equation*}
    Now simply take each of these \(n\) different representations for \(w\) to the \(1/n\)th power to obtain the following list of \(n\)-many \(n\)th roots of \(w\text{.}\)
    \begin{equation*} r^{1/n}e^{i\theta/n}, r^{1/n}e^{i\theta/n + 2\pi i /n}, r^{1/n}e^{i\theta/n + 4\pi i / n}, \dots, r^{1/n}e^{i\theta/n + 2(n-1)\pi i / n} \end{equation*}
    Oh, we could instead have obtained this list by finding the first \(n\)th root of \(w\) and multiplying by all the \(n\)th roots of unity! The \(n\)th roots of \(w = re^{i\theta / n}\) are...
    \begin{equation*} r^{1/n}e^{i\theta / n}, \omega_1r^{1/n}e^{i\theta / n}, \dots, \omega_{n-1}r^{1/n}e^{i\theta / n}, \end{equation*}
    where \(\omega_0,\dots,\omega_{n-1}\) are the \(n\)th roots of unity.

    Example 1.2.14.

    Let \(a\in\R\) be an unspecified real number. We are going to find the two square roots of the complex number \(a+i\) in cartesian form (\(x+iy\)).
    Let's start by defining \(r\) and \(\theta\text{.}\)
    \begin{align*} r \amp = \abs{a+i} = \sqrt{a^2 + 1} \\ \theta \amp = \Arg(a+i) \end{align*}
    So, in polar form, \(a+i\) can be written as \(re^{i(\theta + 2k\pi)}\) for any \(k\in\mathbb{Z}\text{.}\) So we should be able to take \(z_0 = (re^{i\theta})^{1/2} = r^{1/2}e^{i\theta/2}\) and \(z_1 = -z_0\) as our two square-roots. But we still want them in rectangular coordinates, so we use Euler's Formula, some trig identities, and the fact that \(\cos\theta = a/r\text{.}\)
    \begin{align*} z_0 = \sqrt{r} e^{i\theta/2} \amp = \sqrt{r}(\cos(\theta/2) + i\sin(\theta/2)) \\ \amp = \sqrt{r}\left(\sqrt{\frac{1+\cos\theta}{2}} + i \sqrt{\frac{1-\sin\theta}{2}}\right) \\ \amp = \sqrt{r}\left(\sqrt{\frac12 (1+\frac{a}{r})} + i\sqrt{\frac{1}{2} (1 - \frac{a}{r})}\right)\\ \amp = \frac{1}{\sqrt2}(\sqrt{r+a} + i\sqrt{r-a}) \\ \amp = \frac{1}{\sqrt2}(\sqrt{\sqrt{a^2+1}+a} + i\sqrt{\sqrt{a^2+1}-a}) \end{align*}
    The other square-root is \(-z_0\text{.}\)
