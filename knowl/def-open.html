Definition 2.1.1.

For \(\epsilon \gt 0\) and \(p\in\C\) we define
\begin{equation*} B_\epsilon(p) = \set{z\in\C}{ \abs{z - p} \lt \epsilon}, \end{equation*}
the ball or disk of radius \(\epsilon\) centered at \(p\text{.}\)
A set \(U\subseteq \C\) is open if for every \(p\in U\) there is \(\epsilon \gt 0\) for which \(B_\epsilon(p) \subseteq U\text{.}\) A set \(F\subseteq \C\) is closed if its complement \(\C \wo U\) is open.
If you have seen sequential limits before, you should check that \(F\) is closed if and only if every convergent sequence of elements of \(F\) has a limit in \(F\text{.}\)
