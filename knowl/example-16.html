Example 3.2.2.

  1. Let \(f(z) = z^2\text{,}\) so that \(f = u + iv\) where \(u(x,y) = x^2 - y^2\) and \(v(x,y) = 2xy\text{.}\) We know that this function (a polynomial) is differentiable everywhere, so we expect \(u\) and \(v\) to satisfy the Cauchy–Riemann Equations everywhere. Indeed, they do:
    \begin{align*} u_x \amp = 2x = v_y \\ u_y \amp = -2y = -v_x \end{align*}
  2. Consider the conjugation map \(f(z) = \bar z\text{.}\) Then \(f = u + iv\) with \(u(x,y) = x\) and \(v(x,y) = -y\text{.}\) In this case we have \(u_x = 1\) \(v_y = -1\text{,}\) so the Cauchy–Riemann Equations hold nowhere. So the conjugation map is nowhere (complex-)differentiable, which might seem weird since it is a nice orthogonal transformation. In Section 3.5 we will get another explanation: conjugation is not analytic because it fails to preserve angles, in particular their orientation.
  3. The exponential map \(f(z) = e^z\) has real part \(u(x,y) = e^x \cos y\) and imaginary part \(v(x,y) = e^x \sin y\text{.}\) We see that the Cauchy–Riemann equations
    \begin{align*} u_x \amp = e^x \cos y = v_y \\ u_y \amp = -e^x \sin y = -v_x \end{align*}
    hold everywhere, as expected since \(f\) is entire. We also observe that
    \begin{equation*} f'(z) = u_x + i v_x = e^x \cos y + i e^x \sin y = e^z. \end{equation*}
