Solution 6.2.3.1.

As in the previous example, let \(A_R\) be the semicircular path that proceeds from \(R\) through \(Ri\) to \(-R\text{,}\) and let \(C_R\) be the closed path that combines \(A_R\) with the straight-line path from \(-R\) to \(R\text{.}\)
For a non-obvious reason, we will try replacing \(\cos(ax)\) by \(e^{iaz}\) instead of \(\cos(a z)\text{.}\) Use the Residue Theorem to compute the integral over \(C_R\text{:}\)
\begin{equation*} \oint_{C_R} \frac{e^{iaz}}{1 + z^2}\,dz = 2\pi i \Res_{z=i} \frac{e^{iaz}}{1 + z^2} = 2\pi i \left( \frac{e^{iaz}}{2z} \left.\right|_{z = i} \right) = \pi e^{-a} \end{equation*}
Next we deal with the integral over \(A_R\) using an ML estimate. For \(z\) in the upper halfplane,
\begin{equation*} \abs{e^{iaz}} = e^{ia(x + iy)} = e^{ay} \le 1, \end{equation*}
since \(a \gt 0\text{.}\) So the ML estimate can be done almost exactly as in the previous example:
\begin{equation*} \abs{\int_{A_R} \frac{e^{iaz}}{1 + z^2}\,dz} \le \frac{1}{R^2 - 1}\pi R \underset{R\to\infty}{\longrightarrow} 0. \end{equation*}
Just as in the previous example we must have
\begin{equation*} \int_{-\infty}^\infty \frac{e^{iax}}{1 + x^2}\,dx = \pi e^{-a}. \end{equation*}
Recalling DefinitionÂ 4.1.3, we take real parts to get
\begin{equation*} \int_{-\infty}^\infty \frac{\cos(ax)}{1 + x^2}\,dx = \pi e^{-a}. \end{equation*}
(We could take imaginary parts to see that \(\int_{-\infty}^\infty \frac{\sin(ax)}{1 + x^2}\,dx = 0\text{,}\) but that isn't interesting since it is an integral of an odd function over a symmetric interval.)
