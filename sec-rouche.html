    Section 6.3 The Argument Principle and Rouché's Theorem

    Your first step into a larger world...
    A central idea of the Residue Calculus is that the behavior of the singularities inside \(\gamma\) controls \(\int_\gamma f\text{.}\) But we can turn this around: the behavior of \(f\) on \(\gamma\) can give information about the singularities and zeros of \(f\) inside \(\gamma\text{.}\)
    Suppose that \(f\) is analytic on and inside a simple closed curve \(\gamma\text{,}\) oriented counterclockwise, except maybe at finitely many singularities. If \(\oint_\gamma f \ne 0\text{,}\) then there must be at least one singularity.
    Trick: Look at \(\oint_\gamma \frac{f'(z)}{f(z)}\,dz\text{,}\) which equals \(2\pi i \sum \Res \frac{f'(z)}{f(z)}\text{,}\) where the sum is over singularities of \(f'/f\) inside \(\gamma\text{.}\) And this is why the trick works: \(f'/f\) has a singularity at \(a\) means either \(f(a) = 0\) or \(f\) has a singularity at \(a\text{.}\)
    Case 1: \(f(a) = 0\text{.}\) Suppose that \(f\) has a zero of order \(m\text{,}\) so that near \(a\text{,}\)
    \begin{equation*} f(z) = c_m (z-a)^m + c_{m+1}(z-a)^{m+1} + \cdots. \end{equation*}
    So \(f'\) has a zero of order \(m-1\text{:}\)
    \begin{equation*} f'(z) = m c_m (z-a)^{m-1} + (m+1) c_{m+1} (z-a)^m + \cdots . \end{equation*}
    So \(f'/f\) has a simple pole at \(a\text{,}\) and
    \begin{equation*} \Res_{z=a}\frac{f'(z)}{f(z)} = \lim_{z\to a} (z-a) \frac{f'(z)}{f(z)} = \lim_{z\to a} \frac{m c_m (z-a)^m + (m+1)c_{m+1} (z-a)^{m+1} + \cdots}{c_m (z-a)^m + c_{m+1}(z-a)^{m+1} + \cdots} = m. \end{equation*}
    In summary, the residue of \(f'/f\) at \(a\) is the order of the zero of \(f\) at \(a\text{.}\)
    Case 2: \(f\) has a pole of order \(m\) at \(a\text{.}\) Near \(a\) we have a series explansion of the following form.
    \begin{equation*} f(z) = c_{-m} (z-a)^{-m} + \cdots \end{equation*}
    So \(f'\) has a pole of order \(m+1\) at \(a\text{:}\)
    \begin{equation*} f'(z) = -m c_{-m} (z - a)^{-m-1} + \cdots . \end{equation*}
    So \(f'/f\) has a simple pole at \(a\text{:}\)
    \begin{equation*} \Res_{z=a}\frac{f'(z)}{f(z)} = \lim_{z\to a}(z-a) \frac{-m c_{-m}(z-a)^{-m-1} + \cdots}{c_{-m} (z-a)^{-m} + \cdots} = -m. \end{equation*}
    In summary, if \(f\) has no essential singularities inside \(\gamma\text{,}\) then
    \begin{align*} \oint_\gamma \frac{f'(z)}{f(z)}\,dz \amp = 2\pi i \left[ (\text{sum of zeros of } \text{ inside } ) - (\text{sum of poles of } \text { inside } ) \right] \\ \amp = 2\pi i (\text{number of zeros inside } - \text{ number of poles inside } ) \end{align*}
    where the sums count multiplicity.
    OK, but how do we compute \(\oint_\gamma \frac{f'(z)}{f(z)}\,dz\text{?}\) Change variables!
    \begin{equation*} w = f(z), \quad dw = f'(z)\,dz \end{equation*}
    \begin{equation*} \oint_\gamma \frac{f'(z)}{f(z)}\,dz = \oint_{f(\gamma)} \frac{dw}{w} = \text{winding number of around } 0. \end{equation*}
    We have proved the following theorem.

    Theorem 6.3.1. The Argument Principle.

    Suppose that \(\gamma\) is a simple closed curve, oriented counterclockwise. Suppose that \(f\) is analytic on and inside \(\gamma\) except at finitely many poles. Suppose also that \(f\) has no zeros on \(\gamma\text{,}\) either. Then the winding number of \(f(\gamma)\) around \(0\) is
    \begin{equation*} \text{number of zeros of } f \text{ inside } \gamma - \text{number of poles of } f \text{ inside } \gamma, \end{equation*}
    counting multiplicity.

    Example 6.3.2.

    Let \(f(z) = z^5,\) let \(\gamma\) parametrize the unit circle. Inside the unit circle, \(f\) has one quintuple zero and no singularities. The image \(f(\gamma)\) traverses the unit circle \(5\) times and has winding number \(5\text{.}\) \(5 = 5.\)

    Example 6.3.3.

    Try \(f(z) = z^5 + 2\) with the same \(\gamma\text{.}\) The five zeros of \(f\) occur at
    \begin{equation*} 2^{1/5}e^{i \pi / 5}, 2^{1/5} e^{3 \pi i / 5}, 2^{1/5} e^{5 \pi i / 5}, 2^{1/5} e^{7 \pi i / 5}, 2^{1/5} e^{9\pi i / 5}. \end{equation*}
    They all lie outside \(\gamma\text{.}\) Now look at \(f(\gamma)\text{:}\) it's parametrized by \(e^{5i \theta}\text{,}\) \(\theta \in [0,2\pi]\text{.}\) It's the circle \(\abs{z - 1} = 2\) traversed \(5\) times CCW, whose winding number around \(0\) is \(0\text{.}\)

    Example 6.3.4.

    Consider \(f(z) = \frac{1}{z^5}\text{,}\) a function that has \(5\) poles (counting multiplicity) and no zeros inside the unit circle. The image \(f(\gamma)\) is parametrized by \(e^{-5i\theta}\text{,}\) \(\theta \in [0,2\pi]\text{,}\) which winds \(5\) times clockwise around \(0\text{.}\) And \(0 - 5 = -5\text{.}\)

    Example 6.3.5.

    Let's try to count the zeros of \(f(z) = z^5 - z - 1\) inside the circle \(\abs{z} = 2\text{.}\) Parametrize \(\gamma\) as \(\gamma(\theta) = 2e^{i\theta}\text{,}\) \(\theta \in [0,2\pi]\text{.}\) Then \(f\) has no poles, so its number of zeros inside \(\gamma\) is the winding number of \(f(\gamma)\) around \(0\text{.}\)
    \begin{equation*} f(\gamma) = \gamma_1: \quad 32e^{5i\theta} - 2e^{i\theta} - 1, \quad \theta \in [0,2\pi] \end{equation*}
    Let's analyze the simpler \(\gamma_2(t) = 32e^{5i\theta}\text{,}\) \(\theta \in [0,2\pi]\text{.}\) It traverses the large circle \(\abs{z} = 32\) five times CCW, so its winding number around \(0\) is \(5\text{.}\) Let original curve \(\gamma_2\) is pretty close to the original one \(\gamma_1\text{:}\)
    \begin{equation*} \abs{ \gamma_2(\theta) - \gamma_1(\theta)} = \abs{2e^{i\theta} + 1} \le 3 \gg 32. \end{equation*}
    We expect \(\gamma_1\) and \(\gamma_2\) to have the same winding number around \(0\text{...}\)

    Theorem 6.3.6. Walking the Dog.

    If \(\abs{\gamma_2(\theta) - \gamma_1(\theta)} \lt \abs{\gamma_1(\theta)}\) for all \(\theta \in [a,b]\text{,}\) then \(\gamma_1\) and \(\gamma_2\) have the same winding number around \(0\text{.}\)
    In the context of Example 6.3.5, we are saying that \(z^5\) and \(z^5 - z - 1\) have the same number of zeros in \(\abs{z} = 5\text{.}\)

    Theorem 6.3.7. Rouché's Theorem.

    Suppose that \(\gamma\) is a simple closed curve, oriented counterclockwise. Suppose that \(f\) and \(g\) are analytic on and inside \(\gamma\text{.}\) If \(\abs{f(z) - g(z)} \lt \abs{g(z)}\) for all \(z\) on \(\gamma\text{,}\) then \(f\) and \(g\) have the same number of zeros inside \(\gamma\text{.}\)

    Proof.

    Let \(l(z) = f(z) - g(z)\text{.}\) Our assumption is that \(\abs{l(z)} \lt \abs{g(z)}\) for all \(z\) on \(\gamma\text{.}\) We let the “leash” out a little bit at a time: for \(\alpha \in [0,1]\) define
    \begin{equation*} I(\alpha) = \frac{1}{2\pi i} \oint_\gamma \frac{g'(z) + \alpha l'(z)}{g(z) + \alpha l(z)}\,dz \end{equation*}
    Notice that \(I(0)\) equals the number of zeros of \(g\) inside \(\gamma\text{,}\) and \(I(1)\) equals the number of zeros of \(f\) inside \(\gamma\text{.}\) And notice that \(g(z) + \alpha l(z) \ne 0\) because \(\abs{\alpha l(z)} \le \abs{l(z)} \lt \abs{g(z)}\text{.}\) So \(I(\alpha)\) is an integer, and \(I(\alpha)\) varies continuously with \(\alpha\text{,}\) so \(I(0) = I(1)\text{.}\)

    Another proof of the Fundamental Theorem of Algebra.

    Suppose that \(f(z) = a_n z^n + a_{n-1}z^{n-1} + \cdots + a_0\) with \(a_n \ne 0\text{.}\) Set \(g(z) = a_n z^n\text{.}\)
    \begin{align*} \abs{f(z) - g(z)} \amp = \abs{a_{n-1}z^{n-1} + \cdots + a_0} \\ \amp \le \abs{a_{n-1}} R^{n-1} + \cdots + \abs{a_0} \\ \amp \lt \abs{a_n} R^n \end{align*}
    for sufficiently large \(R\text{,}\) because
    \begin{equation*} \frac{ \abs{a_{n-1}}R^n + \cdots + \abs{a_0} }{\abs{a_n}R^n} \underset{R\to\infty}{\longrightarrow} 0. \end{equation*}
    So by Rouché's Theorem, \(f\) and \(g\) have the same number of zeros inside \(\abs{z} = R\text{.}\) And \(g(z) = a_n z^n\) has \(n\) zeros, so \(f\) does too.

    Corollary 6.3.8.

    The modulus of every zero of \(f\) is \(\le \max(1,n\abs{a_{n-1}},\dots,n\abs{a_0})\text{.}\)
