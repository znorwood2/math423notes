Example 4.2.6.

Let's revisit ExampleÂ 4.2.3. We're considering \(f(z) = z^n\) and \(\gamma(t) = e^{it}\) for \(t \in [0,2\pi]\text{.}\) If \(n\ne -1\text{,}\) then \(F(z) = \frac{z^{n+1}}{n+1}\) is an antiderivative of \(f\) on \(\C\text{.}\) So \(\int_\gamma z^n \, dz = F(0) - F(0) = 0\text{.}\)
On the other hand, if \(n = -1\text{,}\) then \(f(z) = \frac{d}{dz} \log(z)\text{,}\) but the complex logarithm can be defined continuously only on a slit plane. In fact, because \(\int \frac{1}{z} \, dz\) does not vanish on the unit circle, there cannot be a continuous branch of the log defined on any domain \(U\) that includes the unit circle. Now we have a proof of that fact!
