    Subsection 5.2.1 Zeros of analytic functions

    Proposition 5.2.1.

    Suppose that \(f\) is analytic on a disk \(B_r(z_0)\) and is not identically \(0\) on that disk. Then there is a unique integer \(k \ge 0\text{,}\) called the order of the zero of \(f\) at \(z_0\text{,}\) such that \(a_k \ne 0\) and \(f\) has Taylor series around \(z_0\) given by
    \begin{equation*} f(z) = (z-z_0)^k (a_k + a_{k+1}(z-z_0) + \cdots) = (z-z_0)^k \sum_{n\ge k} a_n (z-z_0)^{n-k}. \end{equation*}

    Proof.

    This is an immediately corollary to Taylor's Theorem: since \(f\) is not identically zero on \(B_r(z_0)\text{,}\) there is a first nonzero coefficient \(a_k\) in its power series expansion.
    NB. We did not assume in the definition that \(f(z_0) = 0\text{.}\) If \(f(z_0) \ne 0\text{,}\) then \(f\) has a zero of order \(0\) at \(z_0\text{.}\)

    Checkpoint 5.2.2.

    Show how this generalizes the order of a zero of a polynomial from basic algebra. (E.g. the polynomial \((x-2)^3 (x + 5)\) has a zero of order \(3\) at \(x = 2\) and a zero of order \(1\) at \(x = -5\text{.}\))

    Theorem 5.2.3. The zeros of an analytic function are isolated.

    If \(f \colon B_r(z_0) \to \C\) is analytic and not identically zero on \(B_r(z_0)\text{,}\) and \(f(z_0) = 0\text{,}\) then there is a small neighborhood of \(z_0\) in which \(z_0\) is the only zero of \(f\text{.}\)

    Proof.

    As in Proposition 5.2.1, let \(k\) be the order of the zero of \(f\) at \(z_0\text{,}\) and let \(g(z) = a_k + a_{k+1}(z-z_0) + \cdots\) with \(a_k \ne 0\) be the function for which
    \begin{equation*} f(z) = (z-z_0)^k g(z). \end{equation*}
    Then \(g(z_0) = a_k \ne 0\text{,}\) so, since \(g\) is continuous on \(B_r(z_0)\) we can find a small \(\eps \gt 0\) such that \(g(z) \ne 0\) for all \(z \in B_\eps(z_0)\text{.}\) On that small neighborhood we have \(f(z) = (z-z_0)^k g(z) = 0\) iff \(z - z_0\text{.}\)

    Corollary 5.2.4. The Identity Theorem.

    Suppose that \(D\subseteq \C\) is a domain and that \(f\) and \(g\) are two analytic functions \(D \to \C\text{.}\) Suppose further that the set \(S = \set{z \in D}{f(z) = g(z)}\) contains a non-isolated point, that is, a \(w\in S\) such that for all \(\eps \gt 0\) we have \(S \cap B_\eps(w) \ne \{w\}\text{.}\) Then \(f = g\) on \(D\text{.}\)

    Proof.

    Look at the function \(h(z) = f(z) - g(z)\) on \(D\text{.}\) Then \(h\) has a nonisolated zero on \(D\text{,}\) so there is a small disk \(B_\eps(w)\) on which \(h\) is identically zero.
    Now we need a topological argument. Look at the following two sets.
    \begin{align*} U \amp = \set{ a \in D}{h \equiv 0 \text{ on some neighborhood } B_\eps(a) \text{ of } a \text{ in } D} \\ V \amp = \set{ a \in D}{ \text{there is } n \ge 0 \text{ for which } h^{(n)}(a) \ne 0 } \end{align*}
    Certainly \(a \in U\) implies \(a \notin V\text{;}\) i.e., \(U \cap V \ne \emptyset\text{.}\) And \(U \cup V = D\) by Taylor's Theorem: expand around \(a\) and look at the Taylor coefficients. But \(D\) is (path-)connected, so either \(U = \emptyset\) or \(V = \emptyset\text{.}\) We have assumed that \(w \in U\text{,}\) so \(V\) must be empty. Thus \(h\) is identically zero on all of \(D\text{.}\)

    Definition 5.2.5.

    Let \(D_0 \subseteq D\) be domains and \(f_0\colon D_0 \to \C\) analytic. An analytic continuation of \(f_0\) (to \(D\)) is an analytic function \(f\colon D\to \C\) that agrees with \(f_0\) on \(D_0\text{:}\) \(f(z) = f_0(z)\) for all \(z\in D_0\text{.}\)
    An analytic contination of \(f_0\) must be unique, if it exists.

    Example 5.2.6.

    Consider the function
    \begin{equation*} f_0(z) = \sum_{n\ge 0} z^n = 1 + z + z^2 + \cdots \end{equation*}
    defined on \(D_0 = B_1(0)\text{.}\) This series diverges for \(\abs{z} > 1\text{,}\) but we know that \(f(z) = \frac{1}{1-z}\) is an analytic continuation of \(f_0\) defined on \(\C\wo\{1\}\text{.}\)

    Subsection 5.2.2 The Principle of Permanence of Functional Equations

    This is more trouble than it's worth to state precisely, so we'll just give some examples as applications of the Identity Theorem.

    Example 5.2.7.

    The equation \(\sin^2(x) + \cos^2(x) = 1\) holds for all real numbers \(x\text{.}\) This means that we can apply Corollary 5.2.4 to the functions \(f(z) = sin^2(z) + \cos^2(z)\) and \(g(z) = 1\) on \(D = \C\) to conclude that \(sin^2(z) + \cos^2(z) = 1\) for all \(z \in \C\text{.}\)

    Example 5.2.8.

    Consider the equation \(e^{x+y} = e^x e^y\text{,}\) which certainly holds for all real numbers \(x,y\text{.}\)
    The function \(F(z,w) = e^{z+w} - e^z e^w\) is entire in each variable (holding the other one fixed) and equals \(0\) for all \(z,w\in \R\text{.}\) Since \(\R\) has a non-isolated point, this implies by Corollary 5.2.4 that \(F\) is identically zero, i.e., \(e^{z + w} = e^z e^w\) for all \(z,w\in\C\text{.}\)

    Subsection 5.2.3 Singularities

    Definition 5.2.9.

    A function \(f\) is singular at \(z_0\) if \(f\) is not analytic at \(z_0\text{.}\) If \(f\) is singular at \(z_0\) but analytic on \(\set{z }{ 0 \lt \abs{z-z_0} \lt r}\text{,}\) then we say that \(f\) has an isolated singularity at \(z_0\text{.}\)

    Example 5.2.10.

    1. The function \(\frac{z+1}{z^3(z^2+i)}\) has isolated singularities at \(0,\pm i\text{.}\)
    2. The function \(e^{1/z}\) has an isolated singularity at \(z = 0\text{.}\)
    3. The principal branch \(\Log(z)\) of the log has a singularity at \(z = 0\text{,}\) but it is not isolated because of the branch cut.
    4. \(\frac{1}{\sin(\pi/z)}\) has singularities where \(\sin(\pi/z) = 0\text{,}\) in particular at \(z = 0\) and at \(z = 1/n\) for \(n\in\mathbb{Z}\text{.}\) The singularities at \(1/n\) are isolated, but the singularity at \(0\) is not.

    Proposition 5.2.11. Removal of singularities.

    Suppose that \(D\) is a domain and \(z_0 \in D\text{.}\) If \(f\colon D\wo \{z_0\} \to \C\) is analytic and bounded near \(z_0\text{,}\) then \(f\) can be continuously extended to \(D\text{,}\) say \(\lim\limits_{z \to z_0} f(z) = b\text{.}\) In fact, if we define
    \begin{equation*} f^*(z) = \begin {cases} f(z) \amp \text{if } z \in D \wo \{z_0\} \\ b \amp \text{if } z = z_0 \end {cases}, \end{equation*}
    then \(f^*\) is analytic on \(D\text{.}\)

    Proof.

    Define \(h \colon D\to \C\) as follows.
    \begin{equation*} h(z) = \begin {cases} (z- z_0)^2 f(z) \amp \text{if } z \ne z_0 \\ 0 \amp \text{if } z = z_0 \end {cases} \end{equation*}
    Our assumption is that there are \(M\) and \(\eps\) such that \(\abs{f(z)} \lt M\) for all \(z \in B_\eps(z_0) \wo \{z_0\}\text{.}\) Let's estimate the difference quotient of \(h\) near \(z_0\text{:}\)
    \begin{equation*} \abs{ \frac{h(z) - h(z_0)}{z - z_0}} = \frac{\abs{(z-z_0)^2 f(z)}}{\abs{z - z_0}} = \abs{z - z_0} \abs{f(z)} \le \abs{z - z_0} M, \end{equation*}
    which tends to \(0\) as \(z\) tends to \(z_0\text{.}\) So \(h\) is differentiable at \(z_0\text{,}\) in addition to being differentiable near \(z_0\text{.}\) So by Theorem 5.1.2 \(h\) has a power series expansion in a neighborhood \(B_\rho(z_0)\) of \(z_0\text{:}\)
    \begin{equation*} h(z) = \sum_{n\ge 0} a_n (z - z_0)^n = a_0 + a_1(z - z_0) + a_2(z-a_0)^2 + \cdots \end{equation*}
    We must have \(a_0 = a_1 = 0\text{,}\) so we can define
    \begin{equation*} g(z) = \sum_{n\ge 0} a_{n+2} (z-z_0)^n = a_2 + a_3 (z-z_0) + a_4 (z - z_0)^2 + \cdots \end{equation*}
    We have \(g(z) = \frac{h(z)}{(z-z_0)^2} = f(z)\) for \(z \in B_\rho(z_0)\wo \{z_0\}\text{.}\) And \(g(z) \to a_2\) as \(z \to z_0\text{,}\) so \(f(z) \to a_2\) as \(z \to z_0\text{.}\)
    The moral of Proposition 5.2.11 is that the only way a function can fail to be analytic is because it blows up, not because of some weird (real-analysis-like) failure of continuity.
    A function like \(\frac{e^z}{(z-\pi)^5}\) has a singularity at \(z = \pi\text{,}\) and there should be some sense in which the singularity has “order” \(5\text{.}\) The next proposition makes that precise.

    Proposition 5.2.12.

    Suppose that \(D\) is a domain, \(z_0 \in D\text{,}\) and \(f\colon D \wo \{z_0\} \to \C\) is analytic. Suppose also that \(\abs{f(z)} \to \infty\) as \(z\to z_0\text{.}\) Then there are (unique) \(k\in \{1,2,3,\dots\}\) and (unique) analytic \(g\colon D \to \C\) for which \(g(z_0) \ne 0\) and \(f(z) = \frac{g(z)}{(z-z_0)^k}\text{.}\)
    In this case we say that \(f\) has a pole of order \(k\) at \(z_0\text{.}\)

    Proof.

    We begin by constructing \(g\) in a small neighborhood of \(z_0\text{.}\) By our assumption, there is a \(\delta \gt 0\) such that \(\abs{f(z)}\ge 1\) for all \(z\in B_\delta(z_0) \wo \{z_0\}\text{.}\) In particular, \(f\) is nonzero on \(z\in B_\delta(z_0) \wo \{z_0\}\text{.}\) Define \(h\colon B_\delta(z_0) \wo \{z_0\}\) as follows.
    \begin{equation*} h(z) = \begin {cases} \frac{1}{f(z)} \amp \text{if } z \in B_{\delta}(z_0)\wo\{z_0\} \\ 0 \amp \text{if } z = z_0 \end {cases} \end{equation*}
    Then, since \(\abs{f(z)} \to \infty\) as \(z\to z_0\text{,}\) this new function \(h\) is continuous on \(B_\delta(z_0)\) (notice that \(h\) is defined there, since \(f\) is nonzero) and therefore analytic on \(B_\delta(z_0)\) by Removal of Singularities 5.2.11.
    Since \(h(z_0) = 0\) but \(h\) is not identically \(0\) on \(B_\delta(z_0)\text{,}\) this zero must (by Proposition 5.2.1) have an order: there must be \(k\in \{1,2,3,\dots\}\) and an analytic function \(l \colon B_\delta(z_0) \to \C\) for which \(l(z_0) \ne 0\text{.}\) By the continuity of \(l\text{,}\) there is \(\eps \in (0,\delta)\) such that \(l(z) \ne 0\) for all \(z\in B_\eps(z_0)\text{.}\) Define \(g\colon B_\eps(z_0) \to \C\) by \(g(z) = \frac{1}{l(z)}\text{.}\) So \(g\) is analytic on \(B_\eps(z_0)\) and satisfies the following equation for all \(z\in B_\eps(z_0)\text{.}\)
    \begin{equation} g(z) = \frac{1}{l(z)} = \frac{1}{h(z)} \cdot (z - z_0)^k = (z-z_0)^k f(z).\tag{5.2.1} \end{equation}
    But now we can observe that (5.2.1) makes sense for all \(z\in D\text{,}\) so \(g\) has an analytic continuation to all of \(D\text{.}\) So we have found our order \(k\) and function \(g\text{.}\)

    Definition 5.2.13.

    A isolated singularity of \(f\) at \(z_0\) is ...
    • removable if \(f\) is bounded near \(z_0\) (as in Proposition 5.2.11);
    • a pole (of order \(k\)) if it satisfies the hypotheses of Proposition 5.2.12;
    • an isolated essential singularity if it is an isolated singularity that is neither removable nor a pole.

    Example 5.2.14.

    The function \(z\mapsto e^{1/z}\) has an isolated essential singularity at \(z = 0\text{,}\) as we will see.

    Checkpoint 5.2.15.

    Suppose that \(f\) has a zero of order \(m\) at \(a\) and \(g\) has a zero of order \(n\) at \(a\text{.}\) Then \(\frac{f(z)}{g(z)}\) has a “pole of order \(n-m\) at \(a\)”, by which we mean:
    • If \(n \gt m\) then \(f/g\) has a pole of order \(n-m\) at \(a\text{;}\)
    • If \(n \lt m\) then \(f/g\) has a zero of order \(m-n\) at \(a\text{;}\)
    • If \(m = n\) then \(f/g\) is nonzero and analytic at \(a\text{.}\)
    Solution.
    \begin{align} \frac{f(z)}{g(z)} \amp = \frac{c_m (z-a)^m + c_{m+1}(z-a)^{m+1} + \cdots}{d_n (z-a)^n + d_{n+1}(z-a)^{n+1} + \cdots}\notag\\ \amp = \frac{1}{(z-a)^{n-m}} \left( \frac{c_m(z-a)^m + c_{m+1}(z-a)^{m+1} + \cdots}{d_n (z-a)^m + d_{n+1}(z-a)^{m+1} + \cdots} \right)\notag\\ \amp = \frac{1}{(z-a)^{n-m}} \left( \frac{c_m + c_{m+1}(z-a) + \cdots}{d_n + d_{n+1}(z-a) + \cdots} \right)\tag{5.2.2} \end{align}
    Notice that
    \begin{equation*} \frac{1}{(z-a)^{n-m}} = \begin {cases} \frac{1}{(z-a)^{n-m}} \amp \text{if } m \lt n \\ (z-a)^{m-n} \amp \text{if } m \gt n \end {cases}, \end{equation*}
    and that the second factor of (5.2.2) is nonzero at \(z=a\text{.}\)
