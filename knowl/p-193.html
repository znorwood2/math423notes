If \(\abs{f}\) takes the constant value \(0\) on \(D\text{,}\) then \(f\) takes the constant value \(0\) on \(D\text{.}\) So assume from now on that the constant value of \(\abs{f}\) on \(D\) is not zero. Recall that \(\abs{f(z)} = f(z)\bar{f(z)}\text{.}\) Rearrange to see that \(\bar{f(z)} = \frac{1}{\abs{f(z)}^2} f(z)\text{,}\) which means that \(\bar f\) is a constant multiple of the analytic function \(f\text{;}\) therefore \(\bar f\) is analytic.
