Example 4.2.3.

Take the standard parametrization of the unit circle: \(\gamma\colon [0,2\pi]\to\C\text{,}\) \(\gamma(\theta) = e^{i\theta}\text{.}\) Let \(f(z) = z^n\) for an unspecified \(n\in\mathbb{Z}\text{.}\) Let's compute \(\int_\gamma f(z)\,dz\text{:}\)
\begin{equation*} \int_\gamma f(z)\,dz = \int_0^{2pi} (e^{i\theta})^n \cdot i e^{i\theta} \, d\theta = i\int_0^{2\pi} e^{i(n+1)\theta}\,d\theta, \end{equation*}
which equals \(2\pi i\) if \(n = -1\) and \(0\) otherwise.
