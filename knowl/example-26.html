Example 4.1.4.

Compute \(\int_{C_1} \frac{dz}{z}\text{,}\) where \(C_1\) is the top half of the unit circle traversed from \(1\) to \(-1\text{.}\) (We will see below that value of the integral does not depend on the parametrization of the curve, so feel free to use any parametrization of \(C_1\) here.)
The given curve \(C_1\) is parametrized by \(\gamma(t) = e^{it}\) for \(t\in [0,\pi]\text{.}\) We see that \(\gamma(t) = ie^{it}\) and \(f(\gamma(t)) = \frac{1}{e^{it}}\text{.}\) That is all we need to compute the integral:
\begin{equation*} \int_{C_1} \frac{dz}{z} = \int_{t=0}^{\pi} \frac{1}{e^{it}} ie^{it} \,dt = i\int_0^{\pi} dt = \pi i. \end{equation*}
