Lemma 3.6.16.

  1. If \(f\) is a FLT whose only fixed point is \(\infty\text{,}\) then there is a nonzero \(\beta \in \C\) for which \(f(z) = z + \beta\text{.}\)
  2. If \(g\) is a FLT whose fixed points are \(0\) and \(\infty\text{,}\) then there is \(\gamma\ne 1\) for which \(g(z) = \gamma z\text{.}\)
