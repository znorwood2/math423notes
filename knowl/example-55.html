Example 5.3.7.

NB. The Laurent series can depend on the region!
Find the Laurent series for \(f(z) = \frac{1}{z(z-1)}\) around \(a = 0\) for
  1. \(\displaystyle A = \set{ z\in \C }{ \abs{z} \in (0,1) }\)
  2. \(\displaystyle A = \set{ z\in \C }{ \abs{z} \in (1,\infty)}\)
For the first part:
\begin{equation*} f(z) = \frac{-1}{z} \cdot \frac{1}{1-z} = \frac{-1}{z} (1 + z + z^2 + \cdots) = \frac{-1}{z} - 1 - z - z^2 - \cdots \end{equation*}
For the second part, notice that \(\frac{1}{1-z}\) is not convergent on \(A\text{,}\) so we use an algebraic trick using the fact that \(\abs{1/z} \lt 1\) on this \(A\text{:}\)
\begin{equation*} f(z) = \frac{1}{z} \cdot \frac{1}{z(1-1/z)} = \frac{1}{z^2}\left( 1 + \frac{1}{z} + \frac{1}{z^2} + \cdots \right) \end{equation*}
