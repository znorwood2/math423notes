    Section 3.3 Analyticity of inverse functions

    If you paid close attention in calculus, you right remember doing something like this to find the derivative of the logarithm.
    \begin{align*} e^{\log x} = x \amp \Longrightarrow \frac{d}{dx} e^{\log x} = \frac{d}{dx} x \\ \amp \Longrightarrow e^{\log x} \frac{d}{dx} \log(x) = 1 \\ \amp \Longrightarrow \frac{d}{dx} = \frac{1}{e^{\log(x)}} = \frac{1}{x} \end{align*}
    You will be pleased to hear that we can do more-or-less the same thing in complex analysis! The key theorem should say, essentially, that the derivative of \(f^{-1}\) at \((b,f^{-1}(b))\) should be the reciprocal of the derivative of \(f\) at \((a,f(a))\text{.}\) The single-variable calculus version of this fact is illustrated by Figure 3.3.1.

    Theorem 3.3.2.

    Suppose that \(f\) is analytic on a domain \(D\text{,}\) \(z_0 \in D\text{,}\) and \(f'(z_0) \ne 0\text{.}\) There is a disk \(B_\epsilon(z_0) \subseteq D\) on which \(f\) is one-to-one; moreover the image \(V = f[B_\epsilon(z_0)]\) is open, and the inverse function \(f^{-1} \colon V \to U\) is analytic and satisfies
    \begin{equation*} (f^{-1})'(f(z)) = \frac{1}{f'(z)}. \end{equation*}
    We will not bother with a careful proof of Theorem 3.3.2 here. Most of the hard work is handled by the Inverse Function Theorem; the formula Equation  is obtained just as in Calculus by applying the Chain Rule to the equation \(f(f^{-1}(z)) = z\) and solving for \((f^{-1})'(z)\text{.}\)

    Investigation 3.3.1.

    Where does the Inverse Function Theorem come in? you might wonder. Consider a function \(f = u + iv\) that is analytic on a domain \(D\text{.}\) If we think of \(f\) as the function \((u,v) \colon \R^2 \to \R^2\text{,}\) then it has a Jacobian matrix
    \begin{equation*} J_f = \begin {bmatrix} \frac{\partial u}{\partial x} \amp \frac{\partial u}{\partial y} \\ \frac{\partial v}{\partial x} \amp \frac{\partial v}{\partial y} \end {bmatrix} \end{equation*}
    whose determinant can be computed using the Cauchy–Riemann Equations:
    \begin{equation*} \det J_f = u_x v_y - u_y v_x = u_x^2 + v_x^2 = \abs{u_x + i v_x}^2 = \abs{f'}^2. \end{equation*}
    So the Jacobian is zero exactly where the complex derivative is zero. This allows us to state Theorem 3.3.2 in terms of the complex derivative instead of referring to the Jacobian.

    Example 3.3.3.

    The exponential function \(f(z) = e^z\) is entire and has nonzero derivative everywhere. The function Log (the principal branch) is an inverse of \(f\) defined on \(\C \wo (-\infty,0]\text{,}\) and it is analytic on its domain by Theorem 3.3.2. And its derivative is given by
    \begin{equation*} \Log'(z) = \frac{1}{e^{\Log(z)}} = \frac{1}{z}. \end{equation*}
    What's more, other branches of log differ from the principal branch by a constant, so they all have the same derivative!
    Warning. Not every multifunction has the property that all its branches have the same derivative, because not every multifunction has the property that its branches all differ by a constant. Consider arccosine, for example. Or even the square-root, of the next exercise.

    Checkpoint 3.3.4.

    Make sure that you see why Theorem 3.3.2 implies that every branch of the square-root is analytic, and \(\frac{d}{dz} \sqrt z = \frac{1}{2\sqrt z}\text{,}\) where the same square-root function is used on each side of the equation.
