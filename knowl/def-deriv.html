Definition 3.1.1.

A function \(f\colon U\to \C\) defined on a domain \(U\) is differentiable at \(z_0 \in U\) if the following limit exists.
\begin{equation*} \lim_{z\to z_0} \frac{f(z) - f(z_0)}{z-z_0} \end{equation*}
The value of the limit is written \(f'(z_0)\) or \(\frac{df}{dz} (z_0)\text{.}\)
We say that \(f\) is analytic (or holomorphic) at \(z_0\) if there is \(\epsilon \gt 0\) such that \(f\) is differentiable at every \(z\in B_\epsilon(z_0)\text{.}\)
If \(f\) is analytic at every point in \(\C\text{,}\) then we say that \(f\) is entire. (In particular this means \(f\) is defined at every point in \(\C\text{.}\))
