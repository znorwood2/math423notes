This is the idea:
\begin{equation*} \frac{p(z)}{z^n} = a_n + \frac{a_{n-1}}{z} + \frac{a_{n-2}}{z^2} + \cdots + \frac{a_0}{z^n} \underset{\abs{z}\to\infty}{\longrightarrow} a_n. \end{equation*}
Here is a bit more detail. Fixing \(\eps \gt 0\text{,}\) find an \(R \gt 0\) for which the inequality
\begin{equation*} \abs{\frac{p(z)}{z^n} - a_n} = \abs{ \frac{a_{n-1}} + \frac{a_{n-2}}{z^2} + \cdots + \frac{a_0}{z^n} } \lt \eps \end{equation*}
holds for all \(z\) with \(\abs{z} \gt R\text{.}\) Notice that now \(\abs{z} \gt R\) implies \(\abs{\frac{1}{p(z)}} \lt \frac{1}{\abs{a_n} R^n} + \eps\text{.}\) We have found a bound for \(\frac{1}{p(z)}\) for \(\abs{z} \gt R\text{.}\) Now remember that \(z\mapsto \abs{\frac{1}{p(z)}}\) is a continuous function on the compact ball \(\abs{z} \le R\) and is therefore bounded. Combine these two bounds to see that \(\frac{1}{p(z)}\) is bounded on \(\C\text{.}\)
