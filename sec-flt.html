    Section 3.6 Fractional Linear Transformations

    In Example¬†3.5.12 we saw an example of a conformal equivalence. This was in fact our second example of a M√∂bius Transformation, aka Fractional-linear Transformation (FLT).

    Definition 3.6.1.

    A Fractional-linear Transformation (aka FLT aka M√∂bius Transformation) is a function \(f\colon \C\to\C\) of the form
    \begin{equation*} f(z) = \frac{az + b}{cz + d} \end{equation*}
    where \(a,b,c,d\) are complex numbers satisfying \(ad-bc \ne 0\text{.}\)
    (Notice that if we allowed \(ad-bc = 0\) then we'd simply get a constant map. Some authors allow this and call constant maps singular FLTs, so that ours would be non-singular FLTs.)

    Example 3.6.2.

    1. The function from Example¬†3.5.12 is the FLT with \(a = 1\text{,}\) \(b = -i\text{,}\) \(c = 1\text{,}\) and \(d = i\text{.}\)
    2. Complex inversion \(z \mapsto \frac{1}{z}\) is the FLT with \(a = 0\text{,}\) \(b = 1\text{,}\) \(c = 1\text{,}\) and \(d = 0\text{.}\)
    3. The affine transformation \(z \mapsto az + b \) is the FLT with \(c = 0\text{,}\) \(d = 1\text{,}\) and \(a \ne 0\text{.}\)
    It is convenient to regard an FLT \(f(z) = \frac{az+b}{cz + d}\) as a transformation \(\C\cup\{\infty\} \to \C \cup \{\infty\}\) of the extended complex plane by introducing the following conventions.
    If \(c = 0\) then define \(f(\infty) = \infty\text{.}\)
    If \(c \ne 0\) then set \(f(-d/c) = \infty\) and set \(f(\infty) = \frac{a}{c}\text{.}\)
    Where does \(\frac{a}{c}\) come from? Divide the numerator and denominator of \(\frac{az + b}{cz + d}\) by \(z\) and imagine what happens when \(z\) tends toward \(\infty\text{.}\)
    If you have seen some linear algebra, then it is impossible to look at the condition ‚Äú\(ad-bc \ne 0\)‚Äù in the definition of FLT and not wonder whether this has something to do with determinants. We are about to see that it does!

    Proposition 3.6.3.

    1. Every FLT is invertible, and its inverse is another FLT.
    2. The composite of two FLTs is another FLT.

    Proof.

    Consider an FLT \(f(z) = \frac{az+b}{cz+d}\text{.}\) We will show more than what is stated in the Proposition. For the first part, define
    \begin{equation*} g(z) = \frac{1}{ad-bc} \frac{-dz + b}{cz - a}. \end{equation*}
    (Notice that these parameters are exactly the same as the entries of the matrix inverse of the matrix \(\displaystyle \begin {bmatrix} a \amp b \\ c \amp d \end {bmatrix}\text{!}\))
    You can check that \(g(f(z)) = z\text{.}\) (In fact, this follows from the next part of the proof.)
    For part (b), consider FLTs \(f\) and \(g\) given by
    \begin{equation*} f(z) = \frac{az+b}{cz+d} \quad \text{and} \quad g(z) = \frac{pz + q}{rz + s}. \end{equation*}
    You might guess that \(f(g(z))\) corresponds to the product of the matrices \(\displaystyle \begin {bmatrix} a \amp b \\ c \amp d \end {bmatrix}\) and \(\displaystyle \begin {bmatrix} p \amp q \\ r \amp s \end {bmatrix}\text{,}\) and you'd be correct!
    Check that
    \begin{equation*} f(g(z)) = \frac{(ap + br)z + (aq + bs)}{(cp + dr)z + (cq + ds)} \end{equation*}
    and use linear algebra to explain why the new parameters \(A,B,C,D\) satisfy the non-singular condition \(AD - BC \ne 0\text{.}\)

    Investigation 3.6.1.

    What's really going on? The map
    \begin{equation*} \begin{bmatrix} a \amp b \\ c \amp d \end {bmatrix} \mapsto \left(z \mapsto \frac{az + b}{cz + d}\right) \end{equation*}
    defines a group homomorphism \(\operatorname{GL}_2(\C) \to \operatorname{Aut}(\C\cup\{\infty\})\text{.}\) (You don't need to know for this class what these things mean, but you might have seen them in other classes.)

    Theorem 3.6.4.

    Every FLT can be expressed as a composite of dilations (\(z \mapsto az\)), translations (\(z\mapsto z + b\)) and inversions (\(z\mapsto \frac{1}{z}\)).
    ‚ÄúDilation‚Äù is a misnomer, since in the expression \(z \mapsto az\) we allow \(a\) to be any complex number. Remember that multiplication by a fixed complex number \(a\) is really amplification by \(\abs{a}\) followed by rotation by \(\arg(a)\text{.}\)

    Proof.

    Consider \(f(z) = \frac{az + b}{cz + d}\text{.}\) Again, we consider separately the cases where \(c = 0\) and \(c \ne 0\text{.}\)
    If \(c = 0\) then \(f(z) = \frac{a}{d} z + \frac{b}{d} = a(z + \frac{b}{a})\text{.}\) This is the translation \(z \mapsto z+ \frac{b}{a}\) postcomposed by the dilation \(z \mapsto az\text{.}\)
    Now suppose that \(c \ne 0\text{.}\) Divide the numerator and denominator by \(c\) in order to assume \(c = 1\text{:}\)
    \begin{equation*} f(z) = \frac{az+b}{cz + d} = \frac{\frac{a}{c}z + \frac{b}{c}}{z + \frac{d}{c}} = \frac{A z + B}{z + D}. \end{equation*}
    Now do some fraction arithmetic to get
    \begin{equation*} f(z) = A + \frac{B - AD}{Z + D}. \end{equation*}
    Now \(f\) can be written as a composite of translations, inversions, and translations. Look:
    \begin{align*} z \amp \mapsto z + D \amp\amp \text{(translation)}\\ \amp \mapsto \frac{1}{z + D} \amp\amp \text{(inversion)} \\ \amp \mapsto \frac{B - AD}{z + D} \amp\amp \text{(dilation)}\\ \amp \mapsto A + \frac{B-AD}{z+D} \amp\amp \text{(translation)} \end{align*}
    The proof is complete.
    Theorem¬†3.6.4 has the following important geometric corollary.

    Corollary 3.6.5.

    Every FLT is circle-preserving.

    Proof.

    It is intuitively clear (and straightforward to check carefully) that dilations and translations are circle-preserving. And we already proved that the complex inversion map \(z\mapsto 1/z\) preserves circles by considering it as a transformation of the Riemann Sphere!
    Before we move on to other geometric properties of FLTs, we should not forget that we know some calculus! What happens when we differentiate an FLT?
    \begin{equation*} \frac{d}{dz}\left() \frac{az + b}{cz + d} \right) = \frac{(cz+d)(a) - (az+b)(c)}{(cz+d)^2} = \frac{ad - bc}{(cz + d)^2} \end{equation*}
    Notice that this is always nonzero, by our non-singularity condition! Together with the observation that an FLT is analytic everywhere it is defined (it is a rational function!), this amounts to a proof of the following Lemma.

    Lemma 3.6.6.

    An FLT is analytic wherever it is defined, and it is conformal.

    Proposition 3.6.7. Fixed-Point Property of FLTs.

    Every FLT other than the identity map has either one or two fixed points (counting the point at \(\infty\)). That is, there are either one or two solutions to \(f(z) = z\text{.}\)

    Proof.

    As usual, we consider separately the cases where \(c=0\) and \(c \ne 0\text{.}\) Suppose first that \(c = 0\text{,}\) so that our FLT can be written as \(f(z) = az + b\text{.}\) One fixed point is \(\infty\text{.}\) If \(a\ne 1\) then \(z = \frac{b}{1-a}\) is another:
    \begin{equation*} f\left( \frac{b}{1-a} \right) = a\frac{b}{1-a} + b = \frac{b}{1-a}. \end{equation*}
    If \(a=1\) then \(b\ne 0\text{,}\) and so \(z+b = z\) has no solutions other than \(z = \infty\text{.}\)
    Now suppose that \(c \ne 0\text{.}\) Then \(f(\infty) = \frac{a}{c} \ne \infty\text{,}\) so \(\infty\) is not a fixed point. The equation
    \begin{equation*} z = \frac{az + b}{cz + d} \end{equation*}
    can be rearranged into the quadratic equation \(cz^2 - (a-d)z - b = 0\text{,}\) which has two solutions
    \begin{equation*} z = \frac{a-d \pm \sqrt{(a-d)^2 + 4bc}}{2c} \end{equation*}
    if \((a-d)^2 + 4bc \ne 0\text{.}\) Otherwise the two solutions coalesce into just one fixed point: \(\frac{a-d}{2c}\text{.}\)

    Example 3.6.8.

    1. The FLT \(f(z) = \frac{1}{z}\) has two fixed points \(z = \pm 1\text{.}\)
    2. The FLT \(f(z) = \frac{-1}{z}\) has two fixed points \(z = \pm i\text{.}\)

    Corollary 3.6.9. Three Points Property.

    If \(f\) and \(g\) are FLTs with three common values, i.e.,
    \begin{equation*} f(z_1) = g(z_1) \text{ and } f(z_2) = g(z_2) \text{ and } f(z_3) = g(z_3), \end{equation*}
    then \(f = g\text{.}\)

    Proof.

    The function \(g^{-1} \circ f\) (is an FLT by Proposition¬†3.6.3 and) has three fixed points so by Proposition¬†3.6.7 must be the identity map.

    Theorem 3.6.10.

    For all distinct \(q,r,s\) and all distinct \(q', r', s'\) there is a unique FLT \(f\) satisfying \(f(q) = q'\text{,}\) \(f(r) = r'\text{,}\) and \(f(s) = s'\text{.}\)

    Proof.

    Let's start with \(q' = 0\text{,}\) \(r' = 1\text{,}\) and \(s' = \infty\text{.}\) (It is not obvious that this is what we should do, but hang in there and you'll see.)

    Claim 3.6.11.

    For all distinct complex numbers \(q,r,s\) there is a unique FLT \(f\) satisfying \(f(q) = 0\text{,}\) \(f(r) = 1\text{,}\) and \(f(s) = \infty\text{.}\)

    Proof.

    In order for \(f\) to send \(q\) to \(0\) and \(s\) to \(\infty\text{,}\) \(f(z)\) must be proportional to \(\frac{z - q}{z - s}\text{.}\) That is, \(f(z) = k\left( \frac{z-q}{z-s} \right)\) for some \(k\text{.}\) We plug in \(r\) and to solve for \(k\text{.}\)
    \begin{equation*} 1 = r' = f(r) = k\left( \frac{r - q}{r-s} \right), \end{equation*}
    so \(k = \frac{r-s}{r-q}\text{.}\) We conclude that the following FLT \(f\) will do the trick:
    \begin{equation*} f(z) = \left( \frac{r-s}{r-q} \right)\left( \frac{z-q}{z-s} \right). \end{equation*}
    (Remember that \(z\) is the only variable in this expression. So the parameters of \(f\) are e.g. \(b = \frac{r-s}{r-q}(-q)\text{.}\) These have to be interpreted appropriately when you plug in \(\infty\) or get \(0\) in the denominator.)
    By the Claim we can find a FLT \(f\) that sends \(q,r,s\) to \(0,1,\infty\) (respectively) and another FLT \(g\) that sends \(q',r',s'\) to \(0,1,\infty\) (respectively). Now \(g^{-1} \circ f\) sends \(q,r,s\) to \(q',r',s'\text{.}\)

    Definition 3.6.12.

    We define the cross ratio of four complex numbers \(z,q,r,s\) to be
    \begin{equation*} [z : q : r : s] = \left( \frac{r-s}{r-q} \right)\left( \frac{z-q}{z-s} \right), \end{equation*}
    which is the image of \(z\) under the unique FLT sending \(q,r,s\) to \(0,1,\infty\text{.}\)
    Warning: different textbooks use different conventions for the cross ratio.
    It is often useful to have an explicit form for the unique FLT sending \(q,r,s\) to \(q',r',s'\text{.}\) It is given by solving the following equation for \(w\) as a function of \(z\text{,}\) after making appropriate changes to deal with \(\infty\text{:}\)
    \begin{equation*} \frac{(r'-s')(w-q')}{(r'-q')(w-s')} = [w : q' : r' : s'] = [z : q : r : s] = \frac{(r-s)(z-q)}{(r-q)(z-s)}. \end{equation*}

    Example 3.6.14.

    Let's find the unique FLT \(f\) sending \((i,\infty,-i)\) to \((0,1,\infty)\text{.}\) We solve
    \begin{equation*} \frac{w-0}{1-0} = \frac{z-i}{z+i} \end{equation*}
    for \(w\text{.}\) Easy! Get \(w = f(z) = \frac{z-i}{z+i}\text{.}\) Hey, this is the example from Example¬†3.5.12!

    Example 3.6.15.

    Let's find the unique FLT \(f\) sending \((\infty,i,0)\) to \((-1,\infty,1)\text{.}\) We solve the following equation for \(w\text{.}\)
    \begin{equation*} \frac{w+1}{w-1} = \frac{i-0}{1}\left( \frac{1}{z-0} \right) = \frac{i}{z}. \end{equation*}
    We get
    \begin{equation*} f(z) = w = \frac{-1-\frac{i}{z}}{1 - \frac{i}{z}} = \frac{-z - i}{z - i}. \end{equation*}

    Lemma 3.6.16.

    1. If \(f\) is a FLT whose only fixed point is \(\infty\text{,}\) then there is a nonzero \(\beta \in \C\) for which \(f(z) = z + \beta\text{.}\)
    2. If \(g\) is a FLT whose fixed points are \(0\) and \(\infty\text{,}\) then there is \(\gamma\ne 1\) for which \(g(z) = \gamma z\text{.}\)

    Proof.

    Write \(f(z) = \frac{az + b}{cz + d}\text{.}\) Notice that \(c = 0\) since \(\infty\) is a fixed point of \(f\text{.}\) (This follows from how we defined the extension of an FLT to the point at infinity.) So by changing names of constants we can write \(f(z) = az + b\text{.}\) If \(b=0\text{,}\) then \(0\) would be a fixed point of \(f\text{,}\) contradicting our assumption. So \(b\ne 0\text{.}\) If \(a \ne 1\) then \(\frac{b}{1-a}\) would be another fixed point of \(f\text{.}\) So \(a=1\text{;}\) that is, \(f(z) = z + b\text{,}\) as desired.
    For the second part, notice that \(g\) must be the unique FLT that sends \(0\) to \(0\text{,}\) \(\infty\) to \(\infty\text{,}\) and \(1\) to \(g(1)\text{.}\) But \(z \mapsto g(1) z\) also has these properties, so \(g(z) = g(1) z\text{.}\) And \(g(1) \ne 1\text{,}\) since \(g\) has exactly two fixed points (and is therefore not the identity map).

    Lemma 3.6.17.

    FLTs preserve cross ratios. That is, if \(f\) is an FLT then
    \begin{equation*} [f(z) : f(q) : f(r) : f(s)] = [z : q : r : s]. \end{equation*}

    Proof.

    The lefthand side of the equation is \(g(f(z))\) where \(g\) is the unique FLT sending \((f(q),f(r),f(s))\) to \((0,1,\infty)\text{.}\) The righthand side of the equation is \(h(z)\text{,}\) where \(h\) is the unique FLT sending \((q,r,s)\) to \((0,1,\infty)\text{.}\) But \(g \circ f\) is also an FLT sending \((q,r,s)\) to \((0,1,\infty)\text{,}\) so it must be that \(h = g\circ f\text{.}\) In particular, \(h(z) = g(f(z))\text{.}\)
