Solution 6.2.4.1.

Recalling DefinitionÂ 4.1.3, we recognize this integral as a contour integral over the unit circle parametrized by \(\gamma(\theta) = e^{i \theta},\) \(0 \le \theta \le 2\pi\text{.}\) You can think of this as a substitution or change of variables:
\begin{equation*} z = e^{i\theta}, \quad z = iz d\theta, \quad d\theta = \frac{1}{iz}dz \end{equation*}
What remains to replace in the integrand is the \(\cos\theta\text{.}\) For \(\theta\in [0,2\pi]\text{,}\) \(z = e^{i\theta}\) is on the unit circle, so we have
\begin{equation*} \cos\theta = \frac{e^{i\theta} + e^{-i\theta}}{2} = \frac{z + \frac{1}{z}}{2}. \end{equation*}
Now we are ready to do the substitution:
\begin{equation*} \int_0^{2\pi} \frac{1}{a + \cos\theta}\,d\theta = \oint_{\abs{z} = 1} \frac{\frac{dz}{iz}}{a + \frac{z + 1/z}{2}} = \frac{2}{i} \oint_{\abs{z} = 1} \frac{dz}{z^2 + 2az + 1} \end{equation*}
The poles of the integrand (by the Quadratic Formula) occur at \(-a \pm \sqrt{a^2 - 1}\text{.}\) Only \(a + \sqrt{a^2 - 1}\) lies inside the unit circle, so that's the one we need for the Residue Theorem.
\begin{equation*} \Res_{z = -a + \sqrt{a^2 - 1}} \frac{1}{z^2 + 2az + 1} = \frac{1}{2z + 2a} \left.\right|_{z = -a + \sqrt{a^2 - 1}} = \frac{1}{2\sqrt{a^2 - 1}} \end{equation*}
An application of the Residue Theorem finishes the computation:
\begin{equation*} \int_0^{2\pi} \frac{d\theta}{a + \cos\theta} = \frac{2}{i}\cdot 2\pi i \cdot \frac{1}{2\sqrt{a^2 - 1}} = \frac{2\pi}{\sqrt{a^2 - 1}} \end{equation*}
