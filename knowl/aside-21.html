The distance from a point \(z\) to a closed set \(G\) is defined to be the minimum \(\min\set{ \abs{w - z} }{ w \in G}\) when this minimum exists. Notice that if \(z \notin G\text{,}\) then the function \(w \mapsto \abs{w - z}\) is continuous and takes only positive values so must achieve a positive minimum value if \(G\) is compact (= closed and bounded).
