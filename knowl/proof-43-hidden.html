We begin by constructing \(g\) in a small neighborhood of \(z_0\text{.}\) By our assumption, there is a \(\delta \gt 0\) such that \(\abs{f(z)}\ge 1\) for all \(z\in B_\delta(z_0) \wo \{z_0\}\text{.}\) In particular, \(f\) is nonzero on \(z\in B_\delta(z_0) \wo \{z_0\}\text{.}\) Define \(h\colon B_\delta(z_0) \wo \{z_0\}\) as follows.
\begin{equation*} h(z) = \begin {cases} \frac{1}{f(z)} \amp \text{if } z \in B_{\delta}(z_0)\wo\{z_0\} \\ 0 \amp \text{if } z = z_0 \end {cases} \end{equation*}
Then, since \(\abs{f(z)} \to \infty\) as \(z\to z_0\text{,}\) this new function \(h\) is continuous on \(B_\delta(z_0)\) (notice that \(h\) is defined there, since \(f\) is nonzero) and therefore analytic on \(B_\delta(z_0)\) by Removal of Singularities 5.2.11.
Since \(h(z_0) = 0\) but \(h\) is not identically \(0\) on \(B_\delta(z_0)\text{,}\) this zero must (by Proposition 5.2.1) have an order: there must be \(k\in \{1,2,3,\dots\}\) and an analytic function \(l \colon B_\delta(z_0) \to \C\) for which \(l(z_0) \ne 0\text{.}\) By the continuity of \(l\text{,}\) there is \(\eps \in (0,\delta)\) such that \(l(z) \ne 0\) for all \(z\in B_\eps(z_0)\text{.}\) Define \(g\colon B_\eps(z_0) \to \C\) by \(g(z) = \frac{1}{l(z)}\text{.}\) So \(g\) is analytic on \(B_\eps(z_0)\) and satisfies the following equation for all \(z\in B_\eps(z_0)\text{.}\)
\begin{equation} g(z) = \frac{1}{l(z)} = \frac{1}{h(z)} \cdot (z - z_0)^k = (z-z_0)^k f(z).\tag{5.2.1} \end{equation}
But now we can observe that (5.2.1) makes sense for all \(z\in D\text{,}\) so \(g\) has an analytic continuation to all of \(D\text{.}\) So we have found our order \(k\) and function \(g\text{.}\)
