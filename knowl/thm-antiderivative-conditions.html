Theorem 4.3.1.

Suppose that \(f\) is a continuous function defined on a domain \(D\text{.}\) The following are equivalent:
  1. \(f\) has an antiderivative \(F\) on \(D\text{;}\)
  2. for all \(z_1,z_2\in D\) the integral \(\int_{z_1}^{z_2} f(z)\,dz\) is independent of path, meaning that for all contours \(\gamma,\delta\colon [a,b]\to D\) with \(\gamma(a) = \delta(a) = z_1\) and \(\gamma(b) = \delta(b) = z_2\) we have \(\int_\gamma f(z)\,dz = \int_\delta f(z)\,dz\text{;}\)
  3. for all closed paths \(\gamma \colon [a,b]\to D\) we have \(\int_\gamma f(z)\,dz = 0\text{.}\)

Proof.
