Solution 5.4.7.1.

On the annulus defined by \(0 \lt \abs{z} \lt 1\text{,}\) we have the following expression for \(f(z)\text{.}\)
\begin{equation} f(z) = -\frac{1}{z} \cdot \frac{1}{1-z} = -\frac{1}{z}(1 + z + z^2 + \cdots)\tag{5.4.1} \end{equation}
On the annulus defined by \(1 \lt \abs{z} \lt \infty\text{,}\) we have the following expression for \(f(z)\text{.}\)
\begin{equation*} f(z) = \frac{1}{z^2} \cdot \frac{1}{1 - \frac{1}{z}} = \frac{1}{z^2}\left( 1 + \frac{1}{z} + \frac{1}{z^2} + \cdots \right) \end{equation*}
Warning: We must use (5.4.1) to find the residue! The function \(f\) has a simple pole at \(z=0\) with \(\Res\limits_{z=0} f(z) = -1\text{.}\)
