Example 4.5.10.

Compute \(\oint_\gamma \frac{z}{z^2 + 4} \,dz\text{,}\) where \(\gamma\) is the curve pictured below.
Now \(\gamma\) encircles both singularities! So we can't just do what we did in the previous example. So we split the curve in two like this:
Define \(f_1(z) = \frac{z}{z + 2i}\) and \(f_2(z) = \frac{z}{z-2i}\text{,}\) so that
\begin{equation*} f(z) = \frac{z}{z^2 + 4} = \frac{f_1(z)}{z - 2i} = \frac{f_2(z)}{z + 2i}. \end{equation*}
Now we can split up the integral so that we integrate around the two singularities separately:
\begin{align*} \oint_\gamma \frac{z}{z^2 + 4}\,dz \amp = \oint_{\gamma_1 + \gamma_3 - \gamma_3 + \gamma_2} \frac{z}{z^2 + 4}\,dz \\ \amp = \oint_{\gamma_1 + \gamma_2} \frac{f_1(z)}{z - 2i}\,dz + \oint_{\gamma_2 - \gamma_3} \frac{f_2(z)}{z + 2i}\,dz \\ \amp = 2\pi i(f_1(2i) + f_2(2i)) \\ \amp = 2\pi i \end{align*}
It is worth noting that we could've used partial fractions instead: \(\frac{z}{(z-2i)(z+2i)} = \frac{a}{z - 2i} + \frac{b}{z + 2i}\text{.}\) Later, we will see how to do this kind of example much more systematically using residue theory.
