Suppose that \(f(z) = a_n z^n + a_{n-1}z^{n-1} + \cdots + a_0\) with \(a_n \ne 0\text{.}\) Set \(g(z) = a_n z^n\text{.}\)
\begin{align*} \abs{f(z) - g(z)} \amp = \abs{a_{n-1}z^{n-1} + \cdots + a_0} \\ \amp \le \abs{a_{n-1}} R^{n-1} + \cdots + \abs{a_0} \\ \amp \lt \abs{a_n} R^n \end{align*}
for sufficiently large \(R\text{,}\) because
\begin{equation*} \frac{ \abs{a_{n-1}}R^n + \cdots + \abs{a_0} }{\abs{a_n}R^n} \underset{R\to\infty}{\longrightarrow} 0. \end{equation*}
So by Rouch√©'s Theorem, \(f\) and \(g\) have the same number of zeros inside \(\abs{z} = R\text{.}\) And \(g(z) = a_n z^n\) has \(n\) zeros, so \(f\) does too.
